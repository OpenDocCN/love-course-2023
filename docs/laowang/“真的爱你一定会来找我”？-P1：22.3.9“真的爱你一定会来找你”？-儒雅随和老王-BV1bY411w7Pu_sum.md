# 【总结】“真的爱你一定会来找我”？ - P1：22.3.9“真的爱你一定会来找你”？ - 儒雅随和老王 - BV1bY411w7Pu

-   “真的爱你一定会来找我”？
    1.  许多人相信真爱会找到对方，这种信念可能是自我暗示，但并非绝对。
    2.  一个爱你的人也可能不会主动联系你，取决于个人框架和自我控制力。
    3.  对于框架强的人来说，即使很爱你，也可能不会立即联系你，需要更多时间和动力。
-   男人的思考
    1.  男人除了爱与不爱外，还会考虑两人是否能相处好，长期相处糟糕会影响他们的决定。
    2.  在复合过程中，关键是让男人产生希望，觉得两人可以好好相处。
    3.  重要的是放在两人实际相处上，而不是一味要求和责备，相处好了才能让他爱你。
