# 修复感情一定要改变吗？ - P1 - 儒雅随和老王 - BV18B4y1t7JH

多少人跟我说，我要不要改变一切，把自己变成对方完全喜欢的那个样，对于这种偏执的改变，我坚决的说，不，你把自己变成自己讨厌那样，却仅仅是为了挽回对方，那我劝你，你还别这么整了，因为最后就算说你能挽回对方。

你也不可能一直保持一个自己都讨厌的样子，跟人接触，迟早有一天你得后线，你得把那个不好的自己后出来，如果说你总是想，我这样做对方一定喜欢，然后你就不顾自己的感受，等到有一天对方对你说，我喜欢那原来的你。

那你得感受啥，你内心你会不会崩溃，你自己做了这么多，完全不知道对方内心的想法，就好比你觉得对方喜欢温柔的人，你把自己变非常温柔，你从来不发脾气，就算说自己生气的时候，你也不说，你一直就搁那块鞭子。

那对方喜欢善良的人，你就去养一些流汗猫流汗狗来，那要对方说喜欢外表像明星的人，你咋整，你往脸上上点科技，展现真实的你才是最重要的，或许说你身上有很多小毛病，你颜值不高，你不是说富二代，但是最后在一起。

你还是会表现出最真诚的自己，为什么不从一开始就做自己，爱你的人，不会因为你展现出一些小缺点，人就不爱你了，如果说对方因为你这个小缺点，就离开你，那只能说明啥，对方没有那么爱你，你也不必把所有的爱。

花在一个不爱你的人身上，所以说自己是啥样，你就表现出啥样，你别搁那块弄虚作假，你别隐瞒任何的事情，就这样事，然后挽回的时候，才能做到真正的自己。
