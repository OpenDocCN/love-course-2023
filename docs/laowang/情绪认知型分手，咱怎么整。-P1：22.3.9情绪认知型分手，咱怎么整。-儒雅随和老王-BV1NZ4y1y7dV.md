# 情绪认知型分手，咱怎么整。 - P1：22.3.9情绪认知型分手，咱怎么整。 - 儒雅随和老王 - BV1NZ4y1y7dV

首先你得知道人的情绪它是有记忆的，人的情绪它不是悄悄的走，不带走一片云彩，你们因为争吵，他也好你也罢，如果说每次情绪都没有得到疏导，以消化的方式来缓解，那么情绪就会产生记忆，记忆就变成一种情绪人质。

而且不再是单单情绪的本身，我给你举个例子，这个情绪就是啥，有一个不懂事的小孩，每次他耍赖，他躺地下要糖吃，他哭你都不给，次数多了，这小孩他就会慢慢的变得成熟起来，不再跟你耍赖了。

就开始有自己的一个思维了，一段情感，一旦说其中有一个人情绪变得成熟，那意味着啥，他开始变得独立，气焰横扼，他不再那么需要你情绪互动，不再愿意跟你撒娇，不再愿意跟你耍赖，这样基本上，咱就说离了分手。

这就不远了，所以说很多分手，其实不是说需要改变自己和改变对方什么缺点，比起这些日积月费，形成难以改变的这些特质，你现在最急需改变的是啥，对方情绪记忆，以前对方有情绪的时候。

你都按照自己这个方式和思路解决，他的情绪对你就有了一个负面的固化的认知，每当分手之后，对方再次发泄情绪的时候，你就不能再用以前那些技能来应对，而是说你得换一种方式，从未有过的这种正确的方式。

去解决他情绪。
