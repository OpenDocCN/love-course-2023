# 心理学：为什么有些人说不爱就不爱了？ - P1 - 儒雅随和老王 - BV1Bf4y1d7xQ

为啥说有些人他说不爱就不爱了，答案只有一个，你所谓的爱并不是完整意义上的爱，只是浅层次交往范围内的一种深切的喜欢，一个人不可能说不爱他就不爱了，即使说记忆的遗忘都需要很长的时间，把爱的感觉放下来。

它不是一朝一夕的事，如果说一个人他对你说，我说不爱你就不爱，大概就两种可能，一种是他原本就没真的爱上过你，他所说的爱只不过就是谎言，也有可能是啥，就是单纯的为了骗你感情撒谎。

也有可能是他认为自己浅层的喜欢，他就是爱，另一种情况就是啥，他早就不爱你了，只是说你没发现，直到某次爆发之后才被人家点破。
