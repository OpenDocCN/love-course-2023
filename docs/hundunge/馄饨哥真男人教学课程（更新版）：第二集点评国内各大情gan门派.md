# 馄饨哥真男人教学课程（更新版）：第二集点评国内各大情gan门派

好，这道课程我讲一下最大的和以往所有的课程最大一不同点。我们宣传的就三个字，真男人把。什么叫真男人把的？我先来讲一下过去的拔眉方法，包括市场上所有95%的当时的一个拔煤套路。就是他们的把脉套永远这样。

妹子是在这边的，然后男人是在这边的，然后男人永远是根据妹子的一些反应，然后男人是再做出一些反应，然后再去把这个妹子。其实本质上你这样把脉第一个非常耗时间。第二心非常累。第三个你是很难把到这个妹子的。

能听懂我这个意思吗？然后真男人宣传的版倍就是我跟你从一开始从内心深处，我就跟你在同一水平线上，甚至我的这个水平线还比你高这么点高或者高一点点。然后我是来影响这个妹子。

而不是我天天被你这个妹子牵着牵着鼻子走，就这样一种感觉，就只能我这边。然后真男人宣传的不是什么一乱七八糟套路，什么过往什么打呀、冷读啊、推拉，还有什么什么什么技巧。诗歌什么诗格啊，副格啊框架。

我们真男人通通没有这些东西，通通没有，懂吗？我所有的一切东西，真男人就强调三个字，强调他妈四句话，一个是真诚，一个是勇敢。就四个字，真诚勇敢，这是最最主要的，就是你把面一定是一件非常非常快乐的事。

而且是能够坚持做下来的事。那只有就是从内心出发，我要感到我把面是一件很开心快乐。而是我去一天到晚是去我去引导这个妹子，我牵着这个妹子一起走，按照我的游戏规则来走。这样的版位才是可复制，可持续发展。

而不是像现在所有市场上，我这个妹子永远是在这边的，男人永远在这边的，这样你打个麦非常非常累。非常非常的。可以了？小伟哥。你看多久啊，一分钟，这个这个衣服是吧？你那个粘まみ込。
