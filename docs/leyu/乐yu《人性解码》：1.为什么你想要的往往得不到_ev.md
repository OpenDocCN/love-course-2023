# 乐yu《人性解码》：1.为什么你想要的往往得不到_ev

欢迎进入到我们的课程。接下来我会带着大家一起探讨学习，希望在这个课程里能给你有所启发，有所帮助，并且让你真正的感受到由内而外的一种温暖和力量。我相信呢你也有过这种情况，一个人吃完饭，在逛街的时候呢。

看到前面排了很长的一个队伍哈，你会很好奇这些人究竟在排什么。你往前一看，哎，原来是一家甜品店，你听在排队的人说啊，这家店有一款爆款的蛋糕，特别的好吃。于是呢你就跟在队伍后面排队啊，好不容易排了20分钟。

终于买了大家口中说的爆款，你尝了一口以后，发现哎，并不是你很喜欢啊，不是你喜欢的味道，这个时候呢，你把它扔了又觉得浪费，你留着吃嘛，哎，你又觉得有点委屈自己。其实啊在相亲或者是在处对象的过程中啊。

我们也会看到这种情况，你总会遇到一些人哎，感觉还可以。但是在某一方面呢又不达标，你会希望再看看找多几个啊，挑一下有没有更合适的。但最终你也发现很难找到合适又喜欢的那一个对象。那在过去呢。

我做过很多关于脱单的咨询。我听到很多人说，哎呀，我找不到自己喜欢的对象。我都会问他们，你知道自己喜欢什么吗？他会告诉我一大堆看似条件的条件。但在不断深入了解以后啊。

我发现大部分所说的条件其实都非常的模糊。比方说你想找一个啊要对自己好的，但具体什么样才叫对自己好呢？完全的像木头一样听你指挥就叫好吗？又或者是你不能接受欺骗啊。但如果你的另一半是出于要给你惊喜，给你啊。

对你撒了谎，这个时候你能接受吗？当你以为你知道想要什么的时候，你有没有想过，其实这些东西并不清醒，或者是这个东西并不是你想要的。回想一下我们一生当中做了很多的事情，做了很多的选择。

这些事情呢啊这些选择呢有多少件是发自你内心里很想要的，或者跟你的快乐挂钩的呢？有些人呢还容易被别人带偏，被别人影响，做了一些选择啊，最终不是自己喜欢的，或者结果是反的，就容易去怪着别人。哎呀。

你怎么不早跟我说呢？或者是都因为你啊我才会这么做的。一切你觉得别人可以左右你，甚至控制你的都是因为你允许他这么做。当你内心非常坚定，知道想要什么，你是不会听别人说的建议的。就好比我现在让你在手上割几刀。

或者把你的钱都给我吧。不用多想，你肯定不会这么做。但如果我们的内心里模糊不清晰，我们就会希望在外界找到答案。在过去教学的过程中啊，有些人觉得自己不懂怎么跟别人交流，不懂怎么去谈恋爱。

甚至不懂怎么去发信息。但这些所谓不懂的人，他们跟朋友相处，跟朋友发信息是完全没问题的。所以这都是一种错觉。与其说不懂，还更不如说是害怕。害怕这么做会产生负面的结果。那为什么会这个害怕呢？

当你面对一件陌生未曾接触的事情，你会产生一种莫名的恐惧感。好比白天让你在沙滩上走路啊，海风吹过来，哇，好舒服啊。但是如果在深夜的漆黑一片，什么都看不到，让你在沙滩上走的时候，你却会有种哇，完了很恐惧。

很害怕。害怕发信息，害怕谈恋爱，害怕社交其实都是同样的道理，对未知的一种恐惧。变得不再恐惧，唯一的方法就是提高你自身的认知能力。你认知能力越强，消除魔糊的能力也会越强。好比你在登山。

你爬了500米的高度，你发现前面是一片雾霾，伸手不见五指。你代价那你会很彷徨，当你上到了1千米的高度，你发现哦，原来上面是一片艳凉天，雾霾只是你脚底下的一片彩云而已。同理的你的认知越高。

你看到的景象也是不一样。你看清楚了，自然就不再惧怕了。当然哈要提升自己的认知，知道想要什么并不容易。因为我们一生当中会做很多的比较。哦，如果你现在想为家里添置一台投影仪，你会怎么做呢？

你会直接打开网站啊，直接买了吗？还是说你会左挑右挑，哎，看看哪一款啊，性价比高，哪一款功能强大再去入手。那如果你想买一个自行车，是不是你会怎么选啊，你是不是也会通过各种的比较各种的选择。

然后再去做这个购买呢？有一些比较不会对我们产生影响，但有一些比较会让我们心里不舒服。如果你知道马爸爸的身价呢多了100个亿呢，你不会有什么感觉。

但是当你知道小时候比你混的还要差的小伙伴李四啊最近买了一台大奔S500，而你还是在辛苦的打工人，这个时候你心里可能就会有点点不平衡了。其实我们从小到大一直都在做比较，读书的时候比谁读书成绩厉害。

工作了以后比谁的工资高，成家了以后呢，结婚了以后啊，又比谁的家庭幸福比孩子。同学聚会也是各种在比较，一个家庭的老大和老二也是一样的，老二总是希望去模仿老大的行为，或者去抢老大的这个玩具。

其实都是一样的道理。你有没有试过哈，本来想去学习，但是当你打开手机，打开电脑，哎，就变成了刷一下短视频啊，或者是不小心看了一部电影啊，追了一部剧。很容易就被外界啊影响或者吸引，忘了你本来要做的事情。

本质上是一样的，我们身边有太多的诱惑了。当你没有聚焦在你想要的东西的时候，你就会被带偏。我们大部分的眼睛、耳朵都是关注外界的，喜欢看看听听别人说什么，但极少关注或者听听自己内心里的声音。

看看内心你想要什么，但就更加别说去感受自己的真实的感受了。你不懂怎么去表达，因为你没有正视过你内心里的念头，甚至你自己都不叫。感受都不清楚。你内心模糊不清晰时，你会很容易对自己产生怀疑，不相信自己。

但你也会发现一些人呢，他特别的安稳，不受外界影响，他的内心世界是很稳的。一稳就定，这就是内不乱为定。如果你也希望有这种状态，最简单的方法就是门而清。猛人清就是你清楚知道自己的能力边界。

明白自己有什么优缺点。并且呢你知道这些所谓的点都体现在你生活中哪些事情上。当再次发生的时候呢，因为你已经清楚了，你对未知的恐惧就会越来越少，你可以尝试把这些事一一写下来。当你能慢慢感受到自己的时候。

你的状态是饱满的，是丰盈的。你不再需要往外去做对比，不再把焦点放在外头。你清晰知道自己想要什么，明白自己的能力所及，不强求，也就是一种由虚辨实的过程。
