# 乐yu《人性解码》：4.路径：成为一个魅力绽放的人_ev

🎼我们每个人都希望成为一个有吸引力的人。如果你不知道怎么做，你可以先从模仿开始。记得自己在18岁那一年啊，人生中第一次尝试穿西装。在镜子面前呢，哎觉得当时照的时候感觉穿西装就挺成熟的了。

后来发现了男人的成熟跟穿西装跟年龄都没有直接的关系。真正了解到或者接触到西装文化，是看到了007这部电影，一下子被穿西装的邦er迷住了，希望自己也可以成为烂子的男人，于是就各种的去模仿声音。

从声调变成了降调很稳定，说话语是从快到慢，再到日常，西装成为我自己的一个风格。其实这就等于你学画画都会先从临摹大师作品开始。模仿本身就是最快成长的方法，自己也是这么一个过程过来的。再到欣赏和喜欢偶像。

模仿他的穿着和他的行为，他的神情乃至他的口头禅都一一作为参考，商业上也是一样的哈，站在巨人的肩膀上，然后先模仿，再到创作，再到超越。当然了，在成长的过程中，你会听到很多人告诉你应该怎么做。

比方说我们的长辈，他总是会以他们过去的认知，他们的人生观来指导我们。如果你觉得他们的生活方式，或者他们最终的结果是你想要的那你可以吸取他们的建议。那如果并不是你想要的话，你大可以忽略不听。

所以先从模仿开始找到你希望模仿的对象，这是你变得有魅力，有吸引力的一条加速赛道。问过很多人喜欢真花还是假花，得到的答案都是一样的，都说假花虽然完美，没有一点缺陷。真花呢虽然残缺会凋谢，但还是喜欢真花。

因为真花有生命力是那么的真实。是的，我们都很讨厌虚伪的人。但很多时候呢，我们自己却不那么真实，不像小孩子所有的表达都真情流露，肚子饿了，想吃奶了就哭，想跟其他小孩子玩呢，就走过去说，哎呀。

我想跟你一起玩。可是人长大以后啊，都学会了伪装。把自己保护起来，不让别人看穿。哎，觉得被别人知道内心真实的想法，感觉好像就裸露了一样。在我们跟别人交往的过程中啊和相处的过程中。

我们会特别在意别人对我们的看法。比方说你是个女生，你觉得自己的小腿有点粗，你就特意穿长裙子来遮掩你的小腿。如果你介意你的身高，明明你只有167啊，你会说自己170左右。当你都不接受自己那个样子。

你就会更在乎别人对你的看法。事实上，这些点别人可能根本不在意，也不在乎，只有你自己最在意。因为这个世界上在乎你的只有你自己。如果你选择不断撒谎，不断遮掩，不去拥抱和正视你的缺点的话，你会活得很累。

成为真实的自己，因为真实本身就非常的有力量。当然你可能会问哈，如果我有缺点，那我是否正式他就完了？其实正式是第一步，我们要解决一个问题。首先我们得承认有这样的一个问题存在。我也听说过很多人啊。

说我知道我有这样的问题，但我很难改，或者很难取得进展。其实这里的原因很简单，因为你想一次过解决你身上所有的问题啊，并且还想立刻取得结果。其实你根本没必要这么做，你只需要把你最为凸显的几个点先找出来。

每次只解决一个问题。比方说你讲话语速很快，你觉察到了，那你就尝试把它放慢。如果你不敢跟人对视，你就尝试一从短暂的停留开始，慢慢形成习惯，你可以用政治记录法。就是当你改了一次，你给自己画一笔。

看看自己最终能积累多少个政治，每个小的改变都是从你的念头开始，你要注意你的想法，不要觉得细微就不重要了，因为你的想法将会变成你说的话，你说过的话，你会保持一致性选择做同样的行为。

而你的习惯是你很多的行为的重复执行的一个结果。你各种习惯堆积成为了你的性格标签。你的性格将会决定你的命运。就慢慢剔除掉你的劣根性，你才会往完美的方向前进。关注念头，积小成多。到最后你能成为自己的时候。

你也再不需要去做任何的模仿。你只是做回你自己就足够了，你自然会成为你生活中的主角，剔除掉劣根性的过程，也是从生疏到自然，你也不会觉得不舒服，但凡所学皆为性格。
