# 乐yu《人性解码》：11.社交艺术：成为很受欢迎的你_ev

。人是群居动物，我们都脱离不了各种社交关系。有人说他不懂社交，会有社恐。前面已经讲过啊，社恐是因为你不清楚，不知道要做什么，假设你都清楚知道恐惧感就会减少。所以这节课我们一起来聊聊关于社交的艺术。

最好的交流是像朋友一样舒服自然，因为你跟朋友待在一起是不需要刻意去表现，也不需要刻意去得到他们的认可，更不会在意他们对你的看法，因为他们对你已经知根知底了。但在陌生人面前或者在社交中，我们因为有想法。

有需求，所以会在心里形成一种有高有低的状态。其实你应该是对着别人的时候，在一种无需求的状态。因为无需求。所以你的心理也不会有什么得失或者有高低之分，那更好的状态呢，就是当你对着别人有需求。

你还可以做到平等。无高低无差别的对待当你求别人办事儿，你就觉得自己是在低位啊，人家是在高位。但如果反过来别人求你办事的时候呢，哎，你就觉得自己在高位，对方在低位，对着你喜欢的人和对着无所谓的人。

其实也是一样的道理。很多人对着自己喜欢的人就会有一种焦虑感，情式唯唯诺弱。那对着你不在乎的人呢，你会放得很开。不能做到对所有人都一视同仁的心态，你就会随着环境的改变，内心里头也会产生很大的影响。

那关于这个方面呢，我们可以分为几个层次。第一，要把自己当做是对方来看，就是当你站在对方的角度看待事情会有什么样的感受。那第二呢是把对方当做是你自己，这个时候呢，你对他就会像对待自己一样大爱。

因为你是不会让自己吃亏的，你会对自己很无私，所以你也会对他。无私。那第三是把对方看作是他自己本身来对待，就是你接受他本身的样子，而不刻意去改变他。因为你想改变对方的时候呢，你会很痛苦。第四。

就是让你自己做回你自己，你接纳全部接纳自己的好和坏，能包容自己方方面面。因为只有这样你才可以更好的坦然的去接受外部的世界。当你在这种状态下呢，你做的所有事情都会出于内心想做是自愿的。比方说你对一个人好。

你不会因为你对他好，所以你也会希望他可以用同样的方式来回馈你。你仅仅是出于你想对他好而已。因为他值得你这么对待，所以你就这么做了。你的初心，你的出发点，别人是能感受到的。

其实在社交啊在跟别人交流的过程中啊，很多人喜欢去表达，尝试成为焦点。但如果你对这个环境不熟悉，更好的做法是静观其变，去感受当时现场里的每个人的能量，环境的能量，你也可以理解为是感受氛围。

去感受一个环境或者单个人，就像饭局和朋友聚会，总有人是主持方或者是社交中心。那在这个社交中心或者是中心轴，不一定就是说话最多的那个人，有时候也可能是最沉稳的那个人，就比方在饭桌上在酒局上。

大家都会前往向一个人去敬酒。那这个人一定是最有分量的那个人。当你去到一个社交局啊，面对着比较陌生或者完全不认识的人。首先啊你要做的事并不是跟人打交道，打关系，而是让你自己在这个环境里呆着舒服。

让你自己进入一到一个环境里头啊，你能走动了就多走动，看一下感受一下，或者你可以先跟服务员聊聊天，让自己舒服起来。那如果你不能移动的话，那就先从身边最近的那个人开始聊，开始认识。

慢慢去扩大你的社交你的范围。为什么不是先从人开始呢？而是先从环境入手呢？因为事实上并不是每个人你都欣赏，你都会去喜欢所谓圈子不同，不必抢拥。当融入不进去或者不情愿的时候呢。

最起码你可以让自己在整个的社交过程中，哎，在这个环境里头去放松，产生一种很舒服的能量场。要知道啊，宣扬你在一个社交聚会上，当你经历的每一分每一秒都属于你自己的你的每一个时刻都是有意义的那在这个事情上呢。

不在于别人，而是在于你享受了当下，只要你尝试去做，不管什么结果，最起码你的坚NB都是在增长的。总比你一个人躲在角落啊，畏手畏脚，让自己陷入一种很焦虑啊，紧张的状态好。不要想着委屈自己的感受，去迁就别人。

前面已经说过，你的感受比外界重要。如果有机会让你选择的话，那你会选择做个好人，还是选择做个坏人呢？我们都听说过老好人，一般哈都没什么好下场，坏人做了一件好事儿，仿佛他就可以抹掉之前的坏名声。

但是如果好人一旦做了一件坏事，好像就不再值得被原谅了。他身上变成了全是污点，就好比一个渣男，他能回心转意，大家就觉得哇，太难得了，太棒了。但如果一个好男人只是跟另外的人有了一点点的小暧昧。

啊都会被认为是不能接受的，就觉得这个人人格有问题。其实就是坏人有一点点好的表现，就会被放大被加许。斯德哥尔摩综合症就是最好的一个例子。所以啊有时候选择做回你自己，跟随你的内心走，很重要。

哪怕别人觉得你会是一个坏人，因为你的心甘情愿和有所不愿被动选择所做出来的事情，结果是完全不一样的。在感情里头更是这样，你要是活得很卑微，连自己都不尊重自己，那你肯定也不会得到别人的尊重，先知爱。

才能得到别人的爱，建立自己的框架，建立自己的标准和原则。懂得拒绝，懂得 sayay no，这也是一种魅力的表现。🎼。
