# 乐yu《人性解码》：15.不同关系的相处模式_ev

。🎼幸福是什么？幸福就是你跟不同的关系相处，都能得到你想要的结果。你都很安心很舒心。其实我们的生活就是跟不同人打交道，也就是跟不同的关系打交道，由远到近，由深到浅。最能够影响我们的不外乎就是咱们的伴侣。

孩子、父母、朋友，还有你自己。如果让你跟他们这几种关系先后排序，你会怎么排呢？见过很多人会把孩子放在第一位，或者是把父母放在第一位。其实不管你怎么排序啊，都没有对错。只是你把什么关系放在首位。

你自然就会把重点放在那里上面。比方你把孩子放在首位。见到最多的做法就是给孩子过多的关注，以及给予过多的压力。那把父母放在第一位呢，就会很容易成为我们常见的婆媳关系。因为老公总是向着自家母亲。

但几乎很少人会把自己放在所有关系的首位。经过之前所有的课程学习，你应该知道这个世界上你才是最重要的一切都是围绕你来发生。如果你的世界弄丢了，那其他的事情还有什么意义呢？

其实跟我们的关系最亲密的往往也是最能影响我们情绪的人。正所谓，我们把最狠的话都说给了最亲密的人，特别在矛盾争吵的时候，因为你觉得他说的话不好听啊，也不是自己喜欢的。换一个角度看，跟你关系最亲密的人。

他一定是不会希望你坏或者希望你不好的。只是每个人的表达方式有所差异。他们才是真正的希望你好的那个人。比那些只是嘴上说啊，我是爱你的，我是为你好的那些人更值得你去珍惜和拥护。相信再好的伴侣啊。

都有过千百次，心里想掐死对方的念头，相处中总有糟糕的时刻，但这些所谓的好与坏，其实都不是真的，一切都在你的认知里，在你的脑袋里，是你给这些时刻加了定义，主观性的觉得他是好的，或者觉得他是坏的。

其实大部分时间呢，我们都只接受别人好的一面。很难接受别人糟糕的一面，乃至你对自己也都是这样。比方在相处的过程中，我们总想改变对方，希望他可以懂自己跟自己同步。所以形成了一种求同除异。

而并非求同存异的心态。当你想除去对方跟你不一样的地方，你就会苛刻挑刺，还有不接纳。而对方也会以同样的方式来回应你来对待你。但如果你是求同存异，能接受对方不一样的地方。

那你就会抱着开放包容接纳的心态和他相处，对方也会以相同的方式来给你回应。其实真相是什么呢？是你不接纳别人的地方，往往也在你身上存在着。你不接纳的是你自己，你觉得别人身上有的问题，其实这些都是你的问题。

之前我也说过，觉得我妈妈是一个急性子的人，不接纳别人的建议，总想去控制，后来我才发现呢，原来这些其实都是我身上自己的问题。回到你自己身上，想想是不是都是这样的。我们一生当中就希望能找到一个对的人啊。

都在追求完美的伴侣。有个故事是这样的，而子问父亲，我怎么样才可以找到那个对的人了。啊，父亲告诉他，你就忘了这个想法吧，不要想着去找那个对的人，而你应该要成为那个对的人，因为最终你会发现。

你一直在寻找的那个支持你，鼓励你爱你，让你快乐的人，其实一直都在就是你自己。因为你一直想往外去找，其实你是把最重要的东西托付了给别人，把决定权都交到了外界去了。另外啊，如果你本身不是对的人。

哪怕真的幸运让你遇到了那个他遇到的那个认为很对的人。对方一样会离开你，因为你本身就是不对啊，你的水平不够，驾驭不了你的感情也配不上。我有个朋友啊之前遇到一个特别喜欢的男生，觉得对方各方面都好。

思想也跟自己很接近，非常的欣赏这位男生啊对自由的向往，所以就一头的栽进去，觉得这个人非他不可。但在这个过程中呢，我朋友慢慢的给对方很大的压力啊，越这样，这位男生就越想逃离，越逃离。

我朋友就越想把他抓回来，直到最后把对方死死的推开了，最后分手了。其实在这个过程呢，两个人相处都非常的辛苦，就像勒着脖子呼吸一样的感觉。可能听到这里，你会觉得哎，我好像也不太懂得怎么去跟别人相处。

那怎么办呢？老师。真相不是你不懂跟别人相处，而是你连自己都不懂得跟自己相处。你在独处的时候，你都不舒心，你就希望外界给予你这样的感觉。就像我那位朋友一样。所以啊先学会跟自己相处，学会先爱自己。

你才能真正的去爱别人，我跟长得很好看，女生相处过。其实他们对自己也有很多不满意的地方，并没有就是他们外在看起来的那么的自信。所有人都一样，当你觉得自己不够好，你很容易就把焦点放在那些点上。

比方说你觉得自己的腿有点粗，你就想办法啊去遮掩，去遮盖。但你相信这个世界上一定会有人喜欢，可能腿粗一点点的女生。你眼中觉得不好的地方，不一定就是别人认为的缺点。你认为的不一定是你认为的。

之前参加过一个培训班，现场呢刚好来了一对母子，其中一个环节就是聊聊你说过对他最伤害的话。母亲分享了他认为很伤害孩子的那一句。他说你再这样我就不认你了，把你赶出去，他一直觉得很内疚。

那我们现场呢也采访了那个儿子。儿子的记忆中，母亲说过最狠的话，并不是那一句。而是你还算不算一个男人啊，正是这句话深深的伤害了他。其实这也是你认为的不一定就是你认为的。如果你活得很卑微啊。

你自己都不爱自己，一切都是为了别人而活，你生活最终会变得没有任何光彩，如行尸走肉一般。可能你会问，那我这么做，如果拼命的爱自己，会不会变得很自私呢？你要是这么问心慰，你没搞明白什么叫自私，什么叫自爱。

他们的区别。自爱是我好，你好，大家好，我好了，这个世界才会更好。我会希望自己可以变得更加的好。那自私呢就是我好，但我不能够接受你好，我好和你好，其实是冲突的。我不能让你比我更好，我不能接受，这就是自私。

就像情侣分手后啊，你是怀恨在心，你会撕毁对方的照片，想去骚扰他，你见不得他重新进入新的恋爱，见不得他开心快乐，还是说分手以后，你真的发自内心，希望对方好，可以重新的开始新生活，你也希望他可以幸福美满。

最终就是学会跟自己相处，你才是这个世界的中心，不要为了别人活，先学会爱自己，才会让自己发光发亮，自然别人也会被你的光彩照亮。🎼。
