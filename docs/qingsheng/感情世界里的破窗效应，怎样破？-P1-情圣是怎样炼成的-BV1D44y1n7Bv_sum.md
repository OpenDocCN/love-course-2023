# 【总结】感情世界里的破窗效应，怎样破？ - P1 - 情圣是怎样炼成的 - BV1D44y1n7Bv

-   感情世界里的破窗效应解释：破窗效应是指一个问题未及时解决会导致更多问题的产生，如个人逃避困难。逃避问题的行为会导致自信心下降，学习习惯差，成绩下降。
    1.  逃避问题：个人逃避困难，如玩游戏、谈恋爱，导致学习成绩下降。
    2.  学习习惯：缺乏良好学习习惯的人在高中阶段容易出现学习困难。
    3.  自信心受挫：失去成绩作为自信来源后，个人容易怀疑自己能力，导致消极行为。
-   通过考试成绩好等方式获得自信，但若内在支撑崩坏且不修正，会导致自暴自弃，不再追求进步和修正，甚至逃避责任。
    1.  成功领域受挫后，放弃追求，如不再追求进步，会导致自我怀疑和逃避责任。
    2.  自我怀疑时，可能会贬低他人或事物，逃避努力和责任，导致自暴自弃。
    3.  自暴自弃会导致放纵自己，玩乐、恋爱、打架等，缺乏积极修正和进步，破窗效应影响周围人对自己的信心。
