-   自信的定义包括人际交往中的表现、具体问题处理能力、遇到困难时的稳定心态和对待责任的主动性。
    1.  在人际交往中，自信体现在恰当表现自己和处理各种场合的能力。
    2.  处理具体问题时，有经验和能力能增加自信心。
    3.  遇到困难时，稳定心态来源于成就感和信念，能帮助克服困难。
    4.  自信的人对待责任主动，认为责任在自己，有责无旁贷的态度。
-   自责与惭愧心的区别
    1.  勇于承担责任，但不自责
    2.  自责是消极情绪，惭愧心是认错
    3.  自责会让人陷入消极情绪中
-   自信的来源
    1.  自信来自于底气
    2.  底气可能来自于天赋、经验、能力等
    3.  信念和信仰也是自信的来源
