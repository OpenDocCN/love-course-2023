# 【总结】避免聊天中5种尬聊方式，如何高质量沟通 - P1 - 恋爱学长轩哥 - BV19c411J7bG

-   避免尬聊方式
    1.  自說自話，迫切希望把握社交機會，造成對方壓力
    2.  躲避關注式，躲避注意力，緊張不知道說什麼，錯失了機會
    3.  查戶口式提問，集中話題在對方身上，容易讓對方感到煩
-   改善溝通模式
    1.  提問階段，找共同點開啟話題，避免距離感
    2.  誇獎式開場，避免誇外表，多找不同角度誇對方
    3.  傾聽階段，使用語氣詞加入自己的想法，促進對話继续
-   通过高形象的方式让聊天更加兴奋，不同阶段的表达方式可以让对话更加流畅。
    1.  反拋话题，让双方能够聊起来。
    2.  根据话题分享自我或直接反问。
    3.  评论对方，结合自己的情况让对话流动起来。
-   对方的回应至关重要，如果对方冷漠，不要提供太多情绪，尝试几次提问后仍不回答则冷淡。
    1.  如果不知道说什么，可以尝试分享心理学家的爱情36问。
    2.  感情没有捷径，双向奔赴和沟通同样重要。
    3.  保持双向的沟通，共同探讨问题。
