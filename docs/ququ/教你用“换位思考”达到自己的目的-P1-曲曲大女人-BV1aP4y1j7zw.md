# 教你用“换位思考”达到自己的目的 - P1 - 曲曲大女人 - BV1aP4y1j7zw

我给你们写了一个公式叫做情绪加上语言或者行为等于目的，所以说什么意思呢，就是我们一定是在一个什么样的情绪下面，然后呢说出了什么样的话，或者做出来了什么样的事儿，最终达到一个什么样的目的，也就相当于是。

只要我知道了其中任意两项，那么我就可以推导出剩下的那一项，情绪跟语言它是可以包装的，它不一定要是真的，但是我们的目的是不会伪装的，我们要如何达到自己的目的呢。

实际上就是无限的去增加自己脑海当中对于别人情绪，还有这个语言的这个记忆，然后呢知道哦在什么样的情绪表达，什么样的语言表达下，所以能达到这个目的，那么当我们抱守着自己的目的的时候。

我们需要展现什么样的情绪，当你们的语言跟行为是被情绪推动，而不是被目的推动的时候，或者当你们的情绪也不是被目的推动的时候，你们就达不到这个目的，这个就是叫换位思考。

我跟你们表达的东西是全部来源于我自己几十年看到的跟总结的，所以你们肯定不会在别的心理学大师或者说心理的书籍上面看到这个东西。
