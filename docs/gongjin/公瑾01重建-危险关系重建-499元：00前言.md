# 公瑾01重建-危险关系重建-499元：00前言

算是一个前沿吧。我知道每一个同学来到这里学习，目的都是为了解决伴侣出轨的问题。我也知道你们每个人在这个阶段都非常的焦虑，特别希望能够有一个什么样的方法，一下子把出轨这个问题解决了。

更希望能够解决到以后再也不会发生了。你的这个想法我完全能够理解。🎼这是我们学习的共同的目标。但是这个目标不可能一蹴而就，一定是要有一个过程。这个过程可能是半年，可能是一年，总之不会立刻马上就实现。

🎼对于出轨挽回的案例，我服务过的用户大概有2000人。你们在各个阶段的心理以及老公在各个阶段的心里，我也都非常清楚，正因为清楚，所以才有了这样的内容给到大家。

🎼其实呢解决问题最难的地方不是怎么解决你老公出轨的问题，而是解决你的情绪问题。如果你今天的学习是抱着一个立刻马上就解决问题的期待，或者学习之后，马上就想在老公身上看到效果。那我还是要给你泼点冷水。

明确的告诉你，这不可能。如果你抱着一个这样的期待，你的情绪就会不停的反复，你也会一直焦虑，你只要一焦虑，你的行为就会开始踩坑，就会陷入到恶性的循环，那这个问题就不可能解决得了。

🎼我在直播间呢不停的给大家强调，这条路上没有捷径，是希望大家能够正确的看待这个事实。🎼只有你接受了这个事实，你才能够去解决问题。我们学习这门课程呢，也并不都是为了老公。

这个课程的内容呢也不是都教会你如何去挽回老公，而是让你重新去认识婚姻，了解婚姻，了解婚姻中的雷区，了解人性，让你通过学习丰富自己的认知，让你自己变得越来越好。与其说这是一门出轨挽回的课程。

不如说这是一门挽救你的课程。老公出轨固然伤害了你，他的行为是错误的，不道德的，对你有伤害的。这些都是事实。但是这一切事实的背后，是由我们一路参与过来的。我们自己的婚姻出了问题，不可能全部都是别人的问题。

无论这个别人是谁，都没有办法承担全部的责任。我们作为当事人，一定是有责任的。所以不要把自己持续的放到受害者的角色当中来看待这个事情，更不能因为这个事情的发生，就否定自己的。🎼过去和未来，我理解你的痛苦。

但是咱们不能永远的活在痛苦当中。这段痛苦的经历可以给予你力量，让你在这段痛苦的经历当中获得成长，成为更好的你自己，让你自己值得拥有更好的关系，更幸福的人生，成为更坚强、更勇敢、更独立、更快乐的你。

🎼我经常说，离不离婚并不是最重要的。最重要的是你怎么生活，你的生命的质量如何？这些东西不是婚姻决定的，而是你自己决定的。我知道你有很多的束缚，你也有很多的恐惧，但是这一切并不能作为你放弃自己成长的理由。

至少这条路上有我陪着你，你能遇到我就是一种幸运。因为这条路上有太多的坎坷，太多的雷劈，你没有办法独立走完。🎼你选择相信我来跟我学习，我非常愿意搀扶着你，帮助你走出这段人生的至暗时刻。

但是请你再次记住我的话，这条路没有捷径，不要再抱有这样的幻想，请你认真听课，多做笔记，多听直播，有什么问题，及时到学习群里面跟老师提问，或者找助理去约一对一咨询，再或者去到直播间来找我连线。

这是最快的路。如果你不能够把焦点从你的伴侣身上脱离出来。如果你持续的盯着对方，如果你的情绪一直反复，你只能重复上面的步骤，如果你听不进去这些，那就只能放弃治疗。想干什么干点什么吧，想吃点什么。

买点什么吧。珍惜你后边的生命，不要再持续的消耗你自己了，你已经很累了。你的身体已经经受不起你的消耗了。再这样下去，你就要病了。🎼为了这个伤害你的人，持续的折磨你自己，真的很不值得。而且你还有你的亲人。

你的孩子，你的生命不仅仅属于你自己。为了那些爱你的人，那些需要你的人，请你尽快的站起来。这条路上无论是平坦还是崎岖，我和我的团队都愿意给到你支持和帮助。我们跟机构不同。在你身上也没有什么别的所图。

只是期望你能够尽快的好起来，加油吧。亲爱的。
