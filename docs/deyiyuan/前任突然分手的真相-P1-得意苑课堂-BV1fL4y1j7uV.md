# 前任突然分手的真相 - P1 - 得意苑课堂 - BV1fL4y1j7uV

有一个非常扎心的事实，一个人毫无征兆的跟你分手，很大概率都是因为他已经有了下家，只是他跟他的下家还没有稳定关系而已，没有稳定的时候可能还会跟你藕断丝连，当他稳定的时候就一定会对你六亲不认，坚决要分手。

所以现阶段是绝对没有挽回可能性的，或者说很小很小，那如果这个时候你还在苦苦的哀求他，他会连最起码的同情心都没有，甚至会在他的下家那里吐槽你，两个人一起看你的笑话，另一个事实就是。

一旦他和他的下家新鲜感过了，或者说他被下家所伤害，那他大概率会回头来找你疗伤，回头找前任你的麻烦，但如果你欣然接受，又稳定在一起了一段时间，可能你们还是会分手，而且概率在80%，你知道为什么吗。
