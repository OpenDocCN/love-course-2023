# w回的四大阶段 - P1 - 得意苑课堂 - BV14F411475n

往回时一般都需要经历四个阶段，不同的阶段会有不同的特点，也有不同的应对方式，一排斥期，就是两个人刚分手一段时间，一方主动提出了分手，另外一方就算不同意也只能接受，他做出分手的那一刻。

对你的负面印象是非常大的，这个时候你的纠缠不放，只会让他更加厌恶你，如果你想让他回头，就要结束这一种行为，你可以先降低对方的抵触心理，这个时候千万不要主动联系他，更不要在朋友圈里面更新任何的动态。

这段时间你要做的就是专注自己，对于前任，对方没有收到你的消息，也看不到你的动态了，他一定会好奇，你最近这一段时间在做什么，二 展示自我期，经过前一段时间的消失，你引起了对方的好奇。

这个时间就到了展示自我的阶段了，怎么展示呢，利用社交软件展示，如果已经被拉黑，就先想好恢复联系方式，分手之后，你要学会发现生活中的美，扩大自己的社交圈，专注自己的生活，可以跟朋友去旅行。

认识新的异性朋友等等都行，你要让对方知道，你已经开始了新的生活，不再每天都围着他转了，对方看到之后可能会质疑自己，当初分手是不是错了，原来他还是很优秀的，要不要重新跟他在一起呢，三 试探期。

当对方开始在意你的时候，他可能会给你的朋友圈，点赞评论，这时候不要立马回复他，这次是他主动，而你要学会占据有利的局面，对方点赞三次，回应一次就可以了，评论三次回应一次，总之就是让对方觉得你很忙。

一定要把握好自己的节奏，可以冷几天，然后再热几天，同时这段时间，继续在朋友圈展示自己的生活，你要做的就是保持适当的距离感，还有人蜜感，如果在这期间，对方主动找你聊天了，你也不要聊太多，主动的结束话题。

第四 复合期，经过上面的三个阶段了，你们对彼此之间的，已经有了初步的了解，你保持跟他聊天，一直在朋友圈展示自己的改变，你付出了行动，同时他也被你吊足了胃口，现在你们已经达到了，终极对决的时候了。

千万不要随意的放弃跟暴露自己，可以先主动的跟对方见面，通过约会方式，试探一下对方的态度，见面的时候，学会适当的时候，升温你们的感情，比如过马路的时候，顺便牵对方的手，吃饭的时候给对方擦一下嘴巴。

这些暧昧的动作，如果对方没有拒绝，那么你就可以趁机跟对方表明，你们之间的关系，挽回是一个心理博弈的过程，有一个好的心理素质。
