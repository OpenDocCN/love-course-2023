# 【总结】最强约会之“约会要喝点什么？” - P1 - sens开心情感乐园 - BV1UM4y1d7Rg

-   约会时喝什么？
    1.  咖啡类：推荐星巴克、瑞星咖啡、猫舍、犬舍，猫舍犬舍有特色。
    2.  奶茶类：选择周杰伦开的奶茶店、喜茶、拱茶等有特色的奶茶店。
    3.  酒类：在小酒吧点1664百威、可乐桶大无苏等，大无苏比较火，野格也不错。
    4.  行政酒廊：点特调鸡尾酒，接地气简单，不容易出洋相。
    5.  夜店：标准格式是伏特加加野格，便宜且抗喝，搭配果汁或红牛。
-   在夜场点酒要注意选择，香槟是必备，伏特加和野格是主打，不要点啤酒。配饮料时，野格可搭配红牛或果汁，香槟摆在那里增加氛围。建议三四个男的点一瓶野格、两瓶伏特加、十来瓶红牛和果汁，再加一个香槟果盘和纸巾，价格约三四千块钱。 Nightclubs should drink these things.
