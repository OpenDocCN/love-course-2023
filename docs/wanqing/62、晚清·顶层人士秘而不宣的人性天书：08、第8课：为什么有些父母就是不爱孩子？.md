# 62、晚清·顶层人士秘而不宣的人性天书：08、第8课：为什么有些父母就是不爱孩子？

亲爱的小伙伴们，今天我们要分享原生家庭的第二课，为什么有些父母就是不爱自己的孩子？这个标题对有些从小生活在父母宠爱下的年纪又轻的人而言，会非常不适，甚至很不认同。因为我以前就写过不少类似主题的文章。

就经常收到以下这样的留言。比如说哪有父母不爱自己的孩子，坐者你别挑拨离间了好吗？父母肯定爱自己的孩子，天下不会有不爱孩子的父母。😊，每次看到这样的留言，我心情都很复杂，能写下这种留言的人。

他们无疑是幸福的，父母很爱他们，所以才会坚定的认为全天下的父母都是爱孩子的，同时也觉得可悲，为他们可悲，也为那些不被父母所爱的人可悲。😊，那生活在幸福的家庭里，绝对是一件幸运的事儿。但一个人会看书。

会看电视剧，会看新闻，他不是生活在深山老林里的，难道看不见那些被父母卖了，被父母伤害的案例吗？😊，还能说出全天下的父母都爱自己的孩子，那真的是又可悲又可笑，要么认知不健全，要么自欺欺人。

同时更为那些不被父母所爱的同胞可悲。因为有这些天真、认知不健全的所谓的幸福人存在，他们会受到第二步伤害。原本他们不被父母所爱，已经很不幸了，但他们还可以反抗，还可以呼吁。但这些幸福的人会跳出来指责他们。

哪有父母不爱自己的孩子的，你们太不孝了。所以那些不被父母所爱的朋友看到这个标题会感同身受。😊，记得以前我写过一篇广为流传的文章，题目叫这世上有两句最大的谎言，一句叫天下无不失之父母。

一句叫父母做什么都是为了你好。而我也因为文风犀利，观点一针见血，迅速聚集了百万粉丝。今天我们就好好从人性的角度来分析一下，为什么有的父母就是不爱孩子。一般来说，孩子是父母身上掉下来的肉，爱他们。

几乎是本能。从古至今，我们一直都宣称父爱母爱的伟大，也不断强调孝导。但事实上，真正可以称得上伟大的父母并不多，有几个父母不会把自己未尽的梦想转加到孩子身上呢？有几个父母不强求孩子优秀呢？

有几个父母真的尊重孩子的选择呢？有几个父母生养孩子，不求回报，只希望陪伴孩子成长，去参与一段生命的精彩呢？大多都做不到，有的要求孩子光宗耀足，有的要求孩子孝顺听话，有的要求孩子完成自己的梦想。所以。

父母在成为父母之前，他们也不过是普通人，不必用圣人的标准去要求他们。我认为父母有这样那样的要求，也是人之常情，也不必认为父母有这些要求就得自私，毕竟人性本来就是自私的。但还有些父母超越了正常自私的范畴。

他们是真的不爱自己的孩子。那是什么原因造成的呢？其实原因有很多。第一个原因，上一代造的孽，那些不爱孩子的父母固然可恶。但如果我们追溯上去，就会发现他们大多也生活在一个糟糕的原生家庭，不被自己的父母所爱。

他们的父母很少关注他们，冷漠的对待他们，所以他们感受不到来自父母的温暖，时间长了就不知道爱是什么，也不会觉得不爱自己的孩子有什么问题，因为他们的父母也没有爱过他们。所以，原生家庭的问题，如果不去解决。

就会一代代的延续下去。受罪的都是孩子。第二个原因，父母的文化程度不高，思想又特别顽固，有的父母不爱孩子和他们的观念有关，他们的思想特别封建落后，认为孩子是自己的私有物，想打就打，想骂就骂。

甚至觉得你的命是老子给的，老子都有权决定你的生死。关键这样的人你还没法和他沟通，更没法和他讲道理，他脖子一梗，认为我就是这样对待我孩子的，你能拿我怎么办？想想那些家暴诚信的人，大家觉得靠沟通，靠讲道理。

能让他们幡然醒悟吗？其实这种人是无法唤醒的，你只有用更强大的力量去制约他，不敢去做那些事而已。比如他们会天生怕官，怕警察，怕监狱，这就是这类人最典型的人性。第三个原因。

孩子不是父母自愿生或者糊里糊涂生下的。很多人生孩子，并不是因为自己想当父母了，能够担负起一个生命了，喜欢孩子了，而是被各种催促下不得已生的孩子。还有的则是因为不成熟。比如有的姑娘非常恋爱脑。

遇到个渣男就沦陷了，然后再怀孕。这时候渣男不要她了，她也不知道该怎么办。想想看，这些年在宿舍产子在厕所生娃的新闻还少吗？甚至有人把刚出生的孩子生下来就扔到垃圾桶里了，想想都令人愤怒。😊，第四个原因。

父母的生存压力特别大。其实我在很多文章里都提过，如果经济条件不好，生孩子一定要谨慎，但很多妹子会非常生气的质问，难道穷就没有资格生孩子了吗？当然有资格啊，但是你真的考虑过对你自己对孩子好吗？

而我们也会看到，那些条件很好的家庭，往往就剩一个两个，然后精心养育，和那些穷的家徒四壁的夫妻不断的生孩子，3个、4个、5个、6个、7个、一群半大不小的孩子，面黄肌瘦，满目沧桑。

住在那些破的不得了的小房子里。这样的情况下，不能说父母完全不爱孩子，可是条件如此困难，养活这么多张嘴就已经起早贪黑了。他们哪里还有精力去给孩子健康的爱，妈妈的爱呢？在这种情况下长大的孩子。

他们很难内心积极阳光，反而会因为从小生活在困顿中，导致对金钱和物质的极度渴望。第五个原因，父母的经历特别坎坷，内心聚集了太多负能量，导致心理扭曲。有些人的命运很坎坷，一路走来，遭受了很多不公正的对待。

比如被拐卖、被侵犯，或者长期被人欺负，但又没有能力反抗，于是内心渐渐扭曲，因为积累的负能量实在太多了，无处发泄，严格来说，这些人的遭遇也很可怜。他们没有能力去向那些强大的施虐者报复。

于是就把手伸向了毫无反抗能力的孩子。比如有的母亲在被家暴后，他们会带着孩子一起喝农药或者跳楼，他们没有能力去爱自己的孩子，也根本不知道什么叫做尊重一个独立的生命。当然我们这里要讨论的不是这些极端的情况。

遇到这种情况已经不是人性范畴可以解决的问题了。可能有人会痛苦的问，既然他们完全不爱孩子，为什么还要生下来呢？因为人都是自私的。有的人完全不喜欢孩子，但为什么还是生了呢？怕别人议论呢？

所以只能生一个让自己的生活看起来和别人一样。其次，虽然不爱孩子，但是怕自己老了，没人管。死了也没有人送终。为了自己的将来还是生了个孩子出来。所以这个孩子只是他的一个工具而已，虽然他不爱你。

但是从小就会不断和你灌输，要孝顺，服从父母。最可悲的是，这些人不爱孩子，而孩子却在为了得到父母的爱而终身努力。在我做这节课之前，就有一位粉丝找我求助。他说他家在农村，但父母只有他一个孩子。

本来还想继续生孩子的。但是父亲因为酗酒而失去了某种功能，导致生不出其他孩子来了。虽然是家里的独生女，但是父母对她并不好，小时候经常非打即骂，她已经表现的很乖很乖，父母还是稍有不顺心就会打他。

有时候是父亲打他，有时候是父亲找母亲发脾气，母亲心情不好，拿她出气。就这样，他渐渐长大了，开始工作了。从他工作开始，父母就把他90%的工资都拿走了，而他从来不敢买衣服，从来不敢买护肤品，甚至不敢多吃。

这时候有个男的走进了他的生活里，他觉得他很瘦弱，很内向，和人说话也是切切的，就激发了这个男人的保护欲。就这样，两人谈了一年，父母知道了，倒也没有过多反对，主要是父母对他的剥削，左邻右舍的都知道。

看他年纪渐渐大了，父母还漠不关心，明里暗里没少讥讽这对夫妻。没想到这么混蛋的父母居然还会在意左邻右舍的看法，但是他很快就知道为什么父母不反对了，因为父母嫌每个月拿他的工资太不过瘾了，不如直接要一笔彩礼。

而且哪怕他结婚了，难道就敢不给了吗？😊，所以当地彩礼也就三五万，他父母私自大开口要20万，他羞愧难当，但也不知道如何是好。最后，两边经过讨价还价，以16万彩礼成交，和老公结婚后，她内心觉得对不起老公。

害她们家出了这么多彩礼，所以在家包揽了所有家务，对婆婆也非常孝顺。夫家倒是没有为难他，反而觉得他可怜，对他还不错，但好日子没过几天，父母就又找上了他，找他各种要钱。这样的日子持续了3年，老公对她说。

如果继续这样的话，那这婚姻就要走到头了。但每次父母找他的时候，她就害怕的不行，一次次的屈服妥协。于是老公真的提出了离婚，她很痛苦，找我求助。我说，其实你老公对你还是有感情的。

但是她也受不了这种暗无天日的生活。只要你能够做到拒绝父母的无理要求，你是可以挽回老公的。她听了我的话，和老公长谈了一次，老公答应给彼此一个机会，只要他不再无原则的对待父母，那么两人还可以继续过日子。

但是没过几天，父母又找他了，提出要3万块钱出去旅游。如果不给的话，就把他名声搞臭，说他不笑，狼心狗肺。他害怕的来找我问我怎么办？我说你害怕别人的评价是一个方面。

另一个方面你可能自己都没有发现你内心一直在渴求父母爱你，你希望通过你的懂事付出，能换回你父母的爱，你对你父母一直没有死心。他很长时间没有说话，最后才说班琴姐，你可能说的对。

我从小就希望他们能像别的父母对待自己的孩子一样。对我。我很粗心的说，你就算把自己榨干了，你父母也不会以你期望的方式来爱你的，你还是趁早死了这个心，把自己的婚姻保住，不然谁都帮不了你。

我不知道他能不能做到，只是希望他能想明白。所以亲爱的小伙伴们，你发现了没有？有的父母就是不爱孩子的，所以你的父母不爱你，不是你不够好，也不是你的错。你不必因为父母不爱你而怀疑自己，否定自己。

他们爱不爱你，和你本身怎么样，其实并没有太大的关系。你更不必终身耿耿于怀，拼命想让父母来爱你而去做很多事，最后求而不得，痛苦万分。这世上有两种事是无法强求的，一种是明明很爱一个人，却逼着自己不爱。

但有些也是自己的执念造成的。所以在这节课里，我们就要放下自己的执念和自己和解。父母爱不爱你，有时候要看缘分。如果你们没有亲远，也别强求，顶多他们不爱你，你也不爱他们好啦。对别人，包括自己的亲生父母。
