# 【总结】什么是正确的聊天思路 - P1 - 舔狼小吴 - BV19z4y1Y7tR

-   什么是正确的聊天思路
    1.  聊天不需要追求华丽，只要延续对话即可
    2.  不必每句话都华丽，背后可能是故意展示
    3.  不要追求每个案例都很华丽，要适当配合对方
-   聊天内容
    1.  对方可能对你没兴趣，要看对方回复
    2.  延续对话，不要搞错人
    3.  适当客套，约定下次见面
-   男主角在聊天中表现得不上心，导致对方不愿继续聊天，需要反思自己的聊天方式和技巧。
    1.  聊天单调，缺乏情绪，无法自然延续话题。
    2.  缺乏话题感，无法引起对方兴趣。
    3.  需要学会判断何时切换话题，延续对话，以及如何找到主线和支线任务。
