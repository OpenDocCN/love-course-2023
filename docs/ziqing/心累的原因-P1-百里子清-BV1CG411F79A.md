# 心累的原因 - P1 - 百里子清 - BV1CG411F79A

各位，你为什么会感觉到心累？我们感觉到心累，往往是因为这个结果超乎了我们的预期，我们做一件事情，心里是有一个预期的，而这个事情，到了最后这个结果，没有达到我们的预期，然后以结果为导向。

我们就觉得没有达到这种预期，然后心里面有一种失望，失望累积的越多，然后自己就会去怀疑，怀疑自己该不该做这件事情，自己是不是不应该做这件事情，如果我不做这个事情的话，那是不是不会出现这样子的结果，你看。

我们的脑海里面就会想这些事情，那你想这些事情，你不就心累了吗？很多时候就是我们以结果为导向，然后去要求自己去想很多的事情，去计较很多的事情，但是如果我们不顺着这个思路去想，那我接受，我接纳一切的结果。

我只是去做这些事情，我觉得该做的，我就去做，我觉得不该做的，我就不去做，我做了过后要承担它的结果，我不去做也要承担它的结果，而这种结果到底是什么？我都承受，那我选择都承受，我就不会去预估。

我也不会因为这个结果的没达到预期，然后就让自己不高兴，那这样子我就是一种坦然的状态，這樣你還會心累嗎？
