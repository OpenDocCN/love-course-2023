# 坚持每天学习的意义是什么？ - P1 - 百里子清 - BV1514y1M7z7

每天保持一种学习的状态，学习的目的其实就是发现和看到我们自己的不足，很多时候你通过学习你就会发现自己是多么的肤浅，自己是多么的无知，通过学习你才知道这个世界上真的是一山比一山更高。

那我们很多时候只是知道，但是当你真正去学习的时候，你才发现你和别人之间的差距，你比方说我看到了某一个观点，发现我之前为什么没有这样子的想法呢？为什么人家会有这样子深度的思考，而我自己没有呢？

那如此一来我就会看到这一种思考过后，然后别人和自己的这种差距，通过这样子的学习，我就会慢慢的每天去进步一点点，然后自己慢慢的在这种学习过程当中会形成一种习惯，而且会在不知不觉当中让自己得到一种提升。

这种提升它是一种无形的，当某一天你回过头再来看一看的时候，你就会发现自己和之前的那个状态已经是完全不一样了，甚至现在的自己完全是另外一个境界了，所以说每天学习它的这个目的就是不断的鞭策自己。

不断的让我们自己去活得更精彩，活出自己的人生的价值和意义，而很多时候是通过这种学习能够答疑解惑，能够帮我们找到或者说去化解我们心中的一些疑惑，去解决一些问题。

那这样子的话就能够让我们整个的生活充满了一种精彩。
