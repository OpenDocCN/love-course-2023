# 高手都是谋定而后动 - P1 - 百里子清 - BV1rG4y1F7Jt

真正的高手都是看清楚了，想明白了，然后再去执行，而很多人是知道了，刚刚看到了这个东西，然后立马就去执行，这就是高手和普通人之间的差距，高手永远是三思的，永远是要想清楚的，而大多数普通人都是根据自我感觉。

来去做这件事情的，任何一件事情，他有利有弊，高手他会看清楚自己想要的是什么，他会综合判断做这个事情会给自己带来什么，他自己能不能够承受，他会想很多的事情综合考虑过后，看清楚了，想明白了，才去执行。

才去做这件事情，所以你想要成为高手的话，做任何事情之前，请三思，认真地去想一想，认真地思考一番，在我们不了解的情况下，不要妄动，在你不了解的情况下，自己去做一些事情都是错误的，会让自己误入歧途。

而当你陷入那种困境的时候，你才感觉到后悔，那就是无用功了，那就是你走到弯路了，所以真正的高手是先谋定而后动。
