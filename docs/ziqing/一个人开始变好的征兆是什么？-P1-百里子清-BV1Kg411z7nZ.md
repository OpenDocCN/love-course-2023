# 一个人开始变好的征兆是什么？ - P1 - 百里子清 - BV1Kg411z7nZ

各位，一个人开始变好的标志是什么呢？其实开始变好就是我们能够接纳现在的一切，包括你现在的状态，你现在是不好的状态，我们开始去接纳它，承认这些不好，那么你就是慢慢地能够归于一种现实，归于一种平静。

我们都知道要活在当下，但是有很多人都不敢去接受这个当下，我们现在承认这一切，接纳这一切，接纳这个当下，然后再去积蓄能量，当你积蓄能量过后你就会慢慢地变好，而慢慢地变好它是一个过程。

但是如果我们一直想着我该怎么才能变好，那么你永远都变不好，因为你总是在想这个事情，你会去想一些办法，但是没有把时间和精力，用在当下去做一些具体的事情，我们得去做一些具体的事情，承认现在一切的东西。

然后在这个基础上去做一些事情，这个样子逐步的我们才能变好，这是一个过程，它需要时间，也需要我们付出努力，没有一个人的成功，它是随随便便的，但是有很多人，他都是在做一种不切实际的梦，很多人他想变好。

其实是想着我一下子就变好，我一下子就成为有钱人，我一下子就会飞房腾达等等，他是这样子的想法，但是如果是这样子的想法，那只能是白日做夢。
