# 变得更好的途径 - P1 - 百里子清 - BV1SV4y137De

各位，有没有一种方法让自己变得越来越好？如果你的答案是有，那么你会变得越来越糟糕，怎么理解呢？为什么会变成这样？其实我们认真去捋一捋这一个逻辑，你就能够想清楚，能够搞明白了。

当你认为有一种方法可以让自己变得越来越好的时候，你心里面就有更强烈的一个愿望，更强烈的一个心愿，我一定要找到这个方法，那于是我们的时间，精力都会用在这个身上，我们要找到这个方法。

因为我们的逻辑就是我找到这个方法，然后我就会变好了，然后就会顺着这个逻辑去做一些事情，然后就会把自己的时间和精力用在这些上面，但是我们回过头来想一想，你一直把时间和精力用在这些上面，你会慢慢变好吗？

是不可能的，因为要想自己变得越来越好，是要一步一步的脚踏实地的去做一些事情，就是当你放下这种定义的时候，你不要去寻找这种所谓的方法的时候，你顺其自然踏踏实实去做一些具体的事情，那么你在做事情的过程当中。

慢慢的就会变得越来越好，所以很多人他没有去想明白这个道理，你是不是现在头脑也是感觉比较糊涂的，也搞不清楚也搞不懂这到底是因为什么，其实搞不明白那就对了，你再冷静的认真去思考一下。

好好捋一捋你自然而然就有所领悟了，举个例子我想变成一个作家，有没有这种方法呢？当然有了，就是循序渐进的不断的去写，但是如果你总是想着我通过什么样子的方式，一下子就能变成比较厉害的作家，那你能达到吗？

那答案是肯定是否定的，所以我们不要去找什么具体的方式和方法，而是要去做一些事情，做一些事情落到实处，才能成为你想成为的那个人，是这么一个逻辑，那么把这个逻辑理清了，你就知道你接下来该做什么了。
