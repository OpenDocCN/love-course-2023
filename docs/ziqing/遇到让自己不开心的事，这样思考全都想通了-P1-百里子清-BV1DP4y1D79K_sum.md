# 【总结】遇到让自己不开心的事，这样思考全都想通了 - P1 - 百里子清 - BV1DP4y1D79K

-   遇到让自己不开心的事，这样思考全都想通了
    1.  遇到不开心的事情时，想不明白会让人钻牛角尖，进入死循环。
    2.  通过思考宇宙和时间的广度，发现自己渺小，不值得为小事不开心。
    3.  从另一个角度看待不开心的事情，可能有好的一面，提醒我们警惕。
-   通过独立思考和改变观念，我们可以看到世界的不同面貌，学会接受和调整自己的看法。
    1.  感受决定世界的样子，每个人的感受不同。
    2.  学会独立思考，从不同角度看待事情。
    3.  当遇到不开心的事情时，冷静思考，换角度看待，会带来不同的世界。
