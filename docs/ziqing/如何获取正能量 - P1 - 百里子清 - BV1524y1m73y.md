# 如何获取正能量 - P1 - 百里子清 - BV1524y1m73y

怎么获得正能量？其实也很简单，当你不去定义什么是正能量，什么是负能量的时候，那么你就可以获得正能量了，这怎么理解呢？我们来举个简单的例子，我们平常我们每一天脑袋里面，是不是会产生一些负面的情绪。

而这个时候，如果你不承认这种负面情绪的存在，那就相当于我要想办法去消除它，你会发现你越是想办法消除它，你越会增加更多的一个烦恼，而这个烦恼它是不是又是一种负能量，它们累积的越来越多。

这个时候你会更加的烦恼，你的正向能量就会被它们给消耗掉，然后你的正向能量就会慢慢变低，但是如果我承认，我接受这个现实，我允许它的存在，我知道它存在，我接受它，然后我不去做一些事情，那这样子的话。

那就相当于它保持这个状态，而保持这个状态过后，随着时间的一个推移，那整个东西，整个事情都是在变化的，而在这种变化的过程当中，我又慢慢的积蓄能量，积蓄能量过后，我是不是正向的能量，就会慢慢变得越来越多。

所以说你看，我们去承认这一个现实，我们允许负面能量的存在，我不去定义负面能量和正面能量的时候，那么它只是一种存在，所有的东西都是一种存在，我觉得它是什么样子的，它就是什么样子的。

那这样子我们不就获得正能量了吗。
