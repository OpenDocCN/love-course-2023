# 老好人是君子还是小人 - P1 - 百里子清 - BV1814y1p7h6

有粉丝问了这样一个问题，老好人算是君子还是小人？君子还是小人？这个世界上的人，难道都需要这样去区分吗？除了君子就是小人，除了小人就是君子，没有其他的一个区分吗？所谓的老好人，或许应该算是一种伪善之人。

为什么这样讲？因为他的这种做法，他的出发点就是为了让别人知道，他是个好人，他一直所标榜的自己就是这种好，他想得到别人的这种认可，他也觉得自己这样的一个形象，才是自己存在的价值和存在的意义。

但其实他心里面很多时候，面对有的事情的时候，也会恨，也会发怒，但是他不会表现出来，也不敢表现出来，那么表现出来，他的形象就会坍塌，形象坍塌过后，他自己就接受不了，所以他是一种矛盾的存在，他是一个矛盾体。

一方面是尽量的去树立，维护自己的形象，另一方面自己又活得比较压抑，比较纠结，自己一直在给自己较劲，活得不是那么坦然，也不是那么自然，如果从这个角度来讲，那么老好人其实是一个可怜之人。

这种可怜是自己给自己造成的，我们每个人其实都有情绪，我们自己都有爱恨情仇，有的东西你不认可，那就不认可，有的东西你恨那就恨，有的东西你爱那就爱，活得自然一点，不要太在意别人的眼光，那么这样子有可能。

我们就不会去思考这个问题了，老好人到底是小人还是君子，其实和我们没有多大关系。
