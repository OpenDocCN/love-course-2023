# 别人的认可重要吗？ - P1 - 百里子清 - BV1JR4y1m7d9

別人的認可重要嗎？很多人總是挖空心思，想要獲得別人的認可，其實你越是想獲得別人的認可，你得到的是不認可，你得到的是失望，你把更多的時間和更多的希望，寄託在別人身上，就忽略了自己，但是我們每個人。

應該得到的是自己的認可，你獲得別人的認可是一個未知的，而獲得自己的認可，是自己可以去掌握的，是可以自己提升自己的，獲得別人的認可，就是一種未知的東西，就相當於你把所有的希望，都寄託在一個未知數這個上面。

那這樣子你想一想，你每天都會在擔驚受怕之中，你都會在這種焦慮當中度過，而這樣子不就是浪費自己的時間嗎，所以說獲得別人的認可，並沒有我們想像的那麼重要，別人認可你還是不認可你，你還是你。

你能夠想明白這個事情，你就明白了，獲得別人的認可，根本就不是一件，非必要去做的事情，根本就沒有必要去做，這件事情它就是一個無用功，當你冷靜下來，認認真真去思考一下，你自然而然就能發現這一個問題了。

然後你就知道了，我不會去在意別人的認可。
