# 【总结】缺爱的人在恋爱中，为什么很难得到幸福？ - P1：197 - 百里子清 - BV1r34y1771V

-   缺爱的人在恋爱中很难得到幸福
    1.  缺爱的人不知道如何爱自己，缺乏爱他人的能力
    2.  缺爱的人在感情中敏感纠结，缺乏果断和勇气
    3.  遇到问题时，缺爱的人习惯性自我否定，缺乏自信和价值感
    4.  缺爱的人内心孤独，影响性格和恋爱观
    5.  缺爱的人容易降低原则和标准，以错误方式追求爱，难以得到幸福
-   通过经历一些事情，我们成长并改变自己的认知
    1.  随着成长，我们要努力改变缺爱的状态
    2.  改变自己看世界的心态，积极面对生活
    3.  真正的爱是自己给自己的，懂得爱自己才能得到爱。
