# 真正的爱自己是什么样的 - P1 - 百里子清 - BV1Hx4y1G7rT

真正的爱自己是什么样子的情况，那有很多人就会觉得，真正的爱自己就是，我怎么爽就怎么来，我想怎么样就怎么样，其实这样子的一种情况，不是真正的爱自己，真正的爱自己，是要有一种顺其自然的这种状态，就比方说。

我现在伤心了，我难过了，我就要允许自己伤心难过，但是有的人他会觉得，如果要爱自己的话，我就不要去伤心，我也不要难过，因为我要开心，但其实我这个时候就是很伤心，我就是很难过，那你就要接受这个现实。

我现在就要去难过，因为我真的很难过，这种情况你要允许自己难过，就是爱自己，而这个时候你非得要要求自己高兴，那你就是反其道而行之，就是不爱自己，有的时候我们再举个例子，你在遇到一件高兴的事情的时候。

那这个时候你要允许自己去高兴，但是在这个高兴的时候，你要告诉自己，不要太过高兴了，那这个情况下如果说，你一直想保持这种状态，你会说，哎呀我如果爱自己的话，我就是一直要这种开心，一直要这种高兴。

一直这种状态，那就是真正的爱自己，但是其实这个时候，一直的高兴，它是不切合实际的，那么时间久了过后，它也会转入一种平淡，当它转入平淡的时候，我们也接纳平淡，就是说你是个什么样子的状态的时候。

你去接纳那个时候最真实的你，那就是真正的爱自己，我高兴了我就去高兴，我难过了我就去难过，我伤心的时候我就去伤心，我想吃饭了我就想吃饭，我想睡觉了的时候我就睡觉，要允许，要有自然的规律，根据自然的规律来。

而不是我本该去做某件事情的时候，结果我不去做，简单来讲就是让自己融入这种，自然的状态当中，和自然万物融为一体，那这样子才是一种真正的爱自己。
