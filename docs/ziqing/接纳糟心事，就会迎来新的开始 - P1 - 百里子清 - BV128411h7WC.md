# 接纳糟心事，就会迎来新的开始 - P1 - 百里子清 - BV128411h7WC

如果遇到糟心的事情，你的第一反應是什麼？有的人他會順著這個糟心的事情，聯想很多事情，他會想為什麼是我，為什麼我就落到這個境地，為什麼上天偏偏選擇了我，為什麼要讓我承受這些東西，其實你會發現。

當你順著這個思路去想的時候，我們就會陷入一種無止境的煩惱之中，我們就會增加更多的一個煩惱，而在這種情況下，我通常的做法是接納他，要有勇氣去接納他，承認你現在的這種糟心事，承認他的存在，他已經產生了。

他不可能就此消失，而在這個時期，其實我們就相當於處於了，人生的一個低谷期，其實人的一生，它是一個波段式的，有高有低，那這個時間段就相當於，我們處於這個谷底，而在這個谷底，我們學會接納這個谷底，這個現實。

那隨著時間的推移，我的能量會逐漸的積蓄，積蓄能量過後，我就可以朝上走，那好的事情就會轉變過來，就好比我白天，遭遇了很多糟心的事情，我的心情是非常不舒服的，也非常不爽，我也想不明白，但是我現在的選擇是。

不去想他，停止想像，然後晚上好好的去睡一覺，第二天醒來又是一個新的開始，第二天我在回想這個事情的時候，我會發現，其實沒有想像的那麼嚴重，好像昨天某一些煩惱已經忘了，我也不知道當時為什麼那麼煩惱了。

你看這種糟心的事情，我們去接納它，然後就能夠去解決它，但是如果你順著它去思考，然後就會給自己增加更多的煩惱，接納這一切，才有可能創造一個新的開始。
