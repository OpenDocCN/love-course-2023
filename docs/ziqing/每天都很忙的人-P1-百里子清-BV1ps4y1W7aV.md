# 每天都很忙的人 - P1 - 百里子清 - BV1ps4y1W7aV

有些人他其实每天只是看上去挺忙，实际上并没有忙到这个点子上面，所以说从外人的角度来看，好像这个人每天都很忙，这个人都很努力，其实他的这种状态是一种假的，也就是他没有忙在点子上。

没有去思考过我为什么要这样忙，我们每个人其实每一天的时间，精力都是有限的，那你把所有的东西都去做，那你都去忙所有的事情，没有选择性，那这样子的话你就算很忙，也不会有什么效果。

也就是说我们想要去集中精力做一件事情，那么你才能在这件事情上取得一定的成就，而你把自己的时间给分摊了，看上去挺忙，但是没有忙到这个点子上面，那时间久了，你会在这种状态下面，你会觉得有一点自我感动。

好像觉得可以哦，我应该这样子的话会取得好的成就，但是今年哪一月过后你才发现，其实所有的这一切，我们所有的这一些行为，它只是一种假象，自欺欺人的一种假象，所以为什么要学会独立思考。

当你每天感觉都很忙的时候，其实这种状态就是不好的，而这个时候我们需要做的就是停止，不要这么忙，要让自己先停下来，你为什么那么忙，太忙了也是一种错，而这样子的状态是不妥的，是不好的。

那你一直坚持这样子的状态，你会走向何处呢，所以该停的时候就要停一下，太忙的时候就要学会反向思考，自己是不是做事的时候有问题，这个时候需要去认认真真想一想，自己的所作所为，自己的方向。

自己的努力等等是不是有问题了，这个时候不需要再继续保持这种状态，而需要停下来静静独立思考。
