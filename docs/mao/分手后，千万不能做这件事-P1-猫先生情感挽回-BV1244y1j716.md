# 分手后，千万不能做这件事 - P1 - 猫先生情感挽回 - BV1244y1j716

最近我发现很多学员在挽回的过程中最爱做的事情就是写小作文，这时候的作文写得比任何时候都好，也经常会有学员问我老师我发这段话可以吗，暂且不说内容写的是什么。

就从密密麻麻的字数来看就足以给对方很大的压迫感了，而且从那么多年的经验来看，基本上发小作文的就算对方愿意看也只是走马观花的看一遍，最后回你的字数寥寥无几，另外小作文也无法体现出你想要表达的重点。

即便你告诉对方你有多爱他，即便你找出了你们的关系走到分手这一步的根本原因，即便你和对方道歉，但是仅靠区区几百个字对方就能相信你吗，很多机构会让学员去写挽回信，我个人是不建议这么做的。

信息量太大而且会给对方造成心理负担，对方很多时候会觉得你这么来保证，写的这么用心，那我要是再辜负你了承受不起，那长痛还不如短痛，你再想想对方一打开满屏都是你的文字，而且有的时候还需要上翻下滑才能看完。

人的内心都是有限的，更多的你只是感动了自己而不是感动了对方，当然即便是感动了对方，这样换来的挽回也不是长久的，你仔细想想是不是这个道理，我是安琪，你们情感路上的终身陪伴者，如果我的分享对你有用。
