# 你知道前任为什么要和你分手吗？ - P1：3.5号长 - 逢爱情感 - BV19z4y1y7L3

今天我想问一下，你知道你的前任为什么要跟你分手吗，我想这个问题很多人都答不上来，那么我再问你几个问题，那对方的核心需求是什么，你知道吗，假如你是异性，你又愿意和现在的自己在一起吗，想清楚这些问题。

你就会明白对方为什么要跟你分手了，很多时候你们的感情之所以会走向破裂，并不是因为对方不爱你了，而是对方的需求得不到满足，比如说他想要的是苹果，而你就给了他一车的梨，梨虽然好吃，但是呢却不是他想要的。

所以你给他再多也是没有用的，那么你再思考一个问题，你是不是没有用对方喜欢的方式去爱他呢，而是用你自己的方式去爱对方，你明白了吗，你给对方的根本就不是对方想要的，所以你做的再多也只是感动了你自己。

你给对方的爱是不舒服的，那你们的感情又怎么会不破裂呢，记住在爱情里面细致入微的了解，要好过你自以为是的付出，给到对方想要的才是最重要的，你明白了吗，如果还不知道怎么做，找我聊聊，我来帮你。
