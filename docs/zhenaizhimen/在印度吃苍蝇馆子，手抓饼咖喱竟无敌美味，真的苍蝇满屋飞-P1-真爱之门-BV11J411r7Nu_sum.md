# 【总结】在印度吃苍蝇馆子，手抓饼咖喱竟无敌美味，真的苍蝇满屋飞 - P1：191214 在印度吃苍蝇馆子，手抓饼咖喱竟无敌美味，真的苍蝇满屋飞（T） - 真爱之门 - BV11J411r7Nu

-   在印度月光集市购物后，作者独自来到一家街边小饭馆，遇到语言障碍，但尝到了美味的手抓饼咖喱。
    1.  作者在饭馆遇到语言障碍，只能通过手势点菜。
    2.  饭馆环境简陋，但食物美味，特别是手抓饼咖喱。
    3.  作者尝试点了一份饼和咖喱，发现味道十分美味，特别是饼皮酥脆。
-   人们询问是否可以吃某物，得到肯定回答，重复多次。
    1.  人们反复确认可以吃某物。
    2.  人们对吃同样份量的菜花费七百卢比感到不划算。
    3.  人们认为苍蝇馆子比景区实惠。
