# 【总结】你的亲密关系、亲子关系中，为什么会充满了冲突与伤心？ - P1：200413 （T）你的亲密关系、亲子关系中，为什么会充满了冲突与伤心？ - 真爱之门 - BV15Q4y1K7US

-   亲密关系中的纠缠问题
    1.  纠缠导致失去自我
    2.  追求他人的认可和接纳
    3.  无法取悦他人导致自我价值感受受损
-   安吉拉的故事
    1.  努力取悦父亲却无效果
    2.  纠缠导致自我感受受损
    3.  无法取悦丈夫导致亲密关系破裂
-   主人公在舞台上与男人对话，试图让他说出无法取悦的人的话，最终提示他说出"滚蛋"，全场起立鼓掌。
    1.  主人公给男人两次机会，但他仍无法正确回答。
    2.  最终提示男人说出解放性的话语，让他放松并获得掌声。
    3.  主人公强调重拾自己的力量，表达憤怒的艺术，鼓励对每个男人默默说出"滚蛋"。
-   主人公强调重拾自己的力量，不受他人影响，自由表达自己的观点和情感。
    1.  主人公提倡收回自己的权利，不受他人评判、需求和期待的影响。
    2.  主人公强调自由的代价是允许他人自由，接受自己的存在，不受他人干扰。
    3.  主人公强调自由意味着他人有权同意或反对自己的观点，喜欢或不喜欢自己。
-   人们有权选择接受或拒绝他人，自由相处。无论他人对自己有何感受，都与自己无关，评判他人只是内心的反映。不能用期待、恐惧、欲望来控制他人，应保持慈悲和自由。不能评判、控制或操纵他人。
