    2.  困难来自于生活中遇到的陌生情况，使人无法找到心仪的目标或异性。
    3.  走不出去有两种情况，一是不敢往外走，二是因不适感而缩回来，导致无法尝试新事物。

    2.  制造各种困难和意外情况，让运动员适应不适感。
    3.  习惯不适感是取得成功的关键。
-   不适感是学习和成长的必要条件。
    1.  学习新事物时常常会遇到不适感，如学习乐器或研究配方。
    2.  不适感是培养能力和面对挑战的重要途径。
    3.  适应不适感可以帮助人们走出舒适区，取得成功。

    2.  害怕被拒绝以外，更害怕未知和陌生带来的恐惧。
    3.  学习太多知识会减少不适感，建议先适应不适感再学习知识。

    2.  换一个圈子或追求娱乐圈的明星时，害怕程度较高。
