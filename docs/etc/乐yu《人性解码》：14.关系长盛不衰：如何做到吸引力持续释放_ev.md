# 乐yu《人性解码》：14.关系长盛不衰：如何做到吸引力持续释放_ev

。🎼关系相处，永远离不开吸引和追逐。在这节课里呢，我将会带着你去寻找吸引的本质。追和吸引的关系呢其实非常的简单，不主动就是主动，你主动了，你就是被动。举个例子，你用过一款电动牙刷，觉得特别的好用。

聚黑时呢，你向朋友推荐说，我试过用那款牙刷哎，特别的干净，而且非常舒服，反正就各种介绍，推荐他可以买一款试试，真的非常好。过段时间呢你再见到这位朋友，你去问他，哎，上次给你介绍的那个牙刷，哎，怎么样呢？

好用吗？如果你的朋友买都没有买，你觉得他会感谢你给他推荐这个牙刷吗？是不是很像你去追一个人，你去表白或者各种对人家好一样的道理。如果反过来你朋友问你有没有什么好用的牙刷推荐。他想试试电动牙刷。哎。

这时候你轻描淡写的给他推荐那款很棒的牙刷，完了之后呢，你也不需要去问对方怎么样。如果他用过觉得好用啊，觉得不错，他自然会跟你分享说，哎呀，你上次给我推荐了牙刷特别的好，我用了以后呢。

我还给我的朋友推荐了，那你还有没有什么其他好用的东西可以给我分享呢，同样在感情里，当对方被你吸引了，他就会主动想跟你靠近，你也不需要再做过多的行为，还有一点呢吸引应该在关系中一直保持释放。

很多人觉得只要开始的时候在一起有吸引就够了，后面是不需要维系的。所以。为什么很多夫妻啊、情侣啊到最后啊就变得没有了结果，因为不注重自己的吸引力释放。在做很多挽回的咨询中啊，来访者都会意识到。

其实自己并没有好好的打造这一方面。以前帮很多脱单的学生哈，我也经常听到有人跟我说，觉得自己脱不了单啊，不成功了，是因为没有钱，因为自己长得不好看，或者身材不好，所以才得不到对方的青睐。

有人甚至会觉得如果有钱了啊，就可以解决一切的问题，女人就会喜欢自己或对方就喜欢自己。要知道用钱是换不了优质的爱情，就等于用钱是换不了优质的教育。就拿孩子的教育来说，很多人会带孩子参加不同的兴趣补习班。

让孩子可以找一点成长不输同年人，但你会发现这几乎都没什么笑，孩子没学到东西，也没培养出什么兴趣，这是因为你本末倒置了。你该培养的是孩子对一件事情的专注和热爱，并不是各种兴趣。当他有了专注力。

对一件事情感兴趣，他自然就能沉得下去。当然你是他最好的榜样。刚说到啊，钱不是吸引力的关键。因为在感情当中有个道理。你用什么东西吸引人就吸引到什么样的人。用钱可以用钱吸引来的，对方也只会看到你的钱。

因为你就是靠这个东西来吸引别人的呀，当你失去了这个优势以后啊，很可能他对你就没什么兴趣了。如果一个女人是用她的外在来吸引到男人，那这个男人也只会垂涎你的美色啊。

并非那个真实的你我们一生当中啊都在寻找别人的认同，或者别人的认可。小时候努力读书，为了得到爸爸妈妈，得到老师的表扬。出来工作以后，希望得到领导的赏识和表扬，也是一样的道理。那在生活当中和感情中啊。

最常见寻求认可的方式就是去解释，去证明自己是可以的。当你在一个面试的时候呢，为了得到这个职位，你很可能会夸大自己的优势。在处对象时，也因为想跟对方在一起，而刻意强调你们之间有多合拍。

乃至你们有些方面是真的不合适。但反过来，如果你知道自己是有实力的，有能力的，你是不会去解释，就是有机会你见到李嘉诚先生，你刚才说哇，李先生，你真的很有钱了。他是不需要向你证明他的银行卡里有多少个亿。

不需要翻给你看的。蜜蜂不会跟苍蝇解释，为什么花蜜比屎更好，就是一样的道理。不寻求认可的另一种方式是不逞强，不邀功。如果你为伴侣做了顿饭，你得告诉你爸，告诉你妈，告诉全世界还得要发个朋友圈。

仿佛让所有人都知道你做了一件惊天地的事儿，如果你的伴侣并没有给予你期待的回应。你就会产生很大的失落感。而这种失落感呢直接的影响了你的情绪，你也就会带着情绪跟你的伴侣相处。

甚至你还会觉得哎以后我就不给他做饭了。但如果你觉得为伴侣做饭只是一件非常普通的事儿，就像喝了一瓶水一样，你就不会想着去炫耀。你越平常对待另一半就不会感觉到有任何的压力，在没有压力的情况下。

才能很好的享受。那如果在这个过程，他还能自己发现了一些小惊喜，他会更有可能被你感动到。另外啊你越是平常心，越不在乎一件事。期待值越小，你受的。影响程度也越少，因为你不取决于外界给予你的评价。

你不取决于对方回馈你的程度。在感情里，我们都有前任的矛。什么意思呢？就是说你在找对象的时候，很大程度上会参考前任的标准。如果你的前任曾经给你带来很多美好的回忆，你找的对象啊会越接近他。

那如果你的前任给你留下了不可磨灭或者很糟糕的经历啊，那你绝对是不会按着那标准来找的。比方说你被家暴过，或者是你的前任品行很差，或者他抽烟把房子都给烧了，你找对象就一定不能接受这些点。还是开始讲的。

你得清楚知道自己想要什么，因为当你没有标准，你就会在关系中慢慢失去自己，对方的底线就会成为了你的底线。你会一步一步的被吞食。所以啊在一开始相处，就设立标准，这是最起码你得知道自己不能接受什么。

坚守这条底线不能被打破。除了设定标准稳定性也是你吸引力开关很重要的一个因素。记得一次在团建的培训课程中啊，有一个游戏啊，学员跟学员之间发生了一些矛盾，他们不满意游戏的结果。

转而呢就上升到不满意游戏的规则，后来啊就演变成了是学员跟组织方的一些矛盾，那组织方的一些老师呢就解释说，哎呀这个游戏呢其实已经很公平了什么的，大家还是很不满意。后来呢有个老师站出来说。

咱这个游戏呢就是这个设置。没有公平和不公平，你人生中感情里哪有这么多公平的事儿，难道每次不公平不公正的时候，你就去跟你的另一半闹，你就去跟你的公司闹吗？后来大家都被这位老师的话折服。就在这位老师身上。

我看到了一点很难能可贵的地方，遇事不慌，稳定性非常高，不会因为当时的环境和骄躁的情绪所影响，其实在感情中啊，有些人会出于自我保护啊，或者是自我满足，像你有意识无意识的发起考验。

他想看看你是否有能力扛住这些情况。假设你能稳住，哎，就通过考验。假设没稳住，他会觉得你是个不能依靠，不能依赖的人，所以处理这些事情上还是得遵从大原则，不要被表象迷惑影响，稳中求胜。

因为我们很容易会被对方的评价影响情绪。比方人家跟你说，哎呀，你这个鞋子或者你这个衣服不好看，跟你完全不搭。听完后呢，你可能就会去解释说他我这个衣服啊好贵的，或者我这个鞋子是在哪里买的。😊。

其实这些都完全没必要。如果这些话是从一个几岁的小孩子嘴中说出来，就所谓的童言无忌，你还会这么在意吗？同样的情景，同样的话，所有的咀决于我们内心世界怎么看待它。所以在吸引力层面呢，要明白背后的逻辑。

不需要再去得到别人的认可，不需要再去向别人证明自己，你要有自己的标准，增强自身的稳定性。🎼。
