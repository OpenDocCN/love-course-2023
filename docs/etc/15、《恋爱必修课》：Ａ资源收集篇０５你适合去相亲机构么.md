# 15、《恋爱必修课》：Ａ资源收集篇０５你适合去相亲机构么

你好，我是俊瑶，欢迎收听俊瑶的恋爱必修课。那今天我们来聊一聊相亲机构，也就是婚介中心。我在之前的很多视频里边说过婚介中心的一些内幕，然后很多的从业人员会来评论或者私信，无外乎是一些比较恶毒的话。

或者说我小人之心度君子之父之类的。我点开看了一下呀，我也确实相信他们所说的，他们机构没有我所讲的那些问题，是因为他们的机构实在是太小了，就好像邻居张大妈给同事的儿子介绍一个自己的外甥女儿。

这个肯定没什么内幕，那，最多就是吹吹牛而已。但是一旦机构做大了，难免下面的员工会有一些自作主张的这种小手段。那这一节课主要是给你去做一些避坑指南。就好像大家在网上看测评一样。

介绍一下婚介机构的服务是不是适合你能不能帮到你。我对这些机构的态度其实是很好的，我没什么。面情绪甚至会推荐给身边的一些合适的需要的人去，只是经常会有粉丝过来给我讲，在里边遇到的坑。

所以我想和大家分享一下啊，篇幅有限，给大家说两个最重要的事情。第一个事情是收费标准。大部分机构都很少会给你一个非常明确的加目表，而是给你一个大概的标准，实际价格其实是看销售报价。

而报价的基础是看你有多少钱，以及你的要求有多么的离谱。你有多少钱，这个好理解啊，因为本身就是一个非标准化的服务产品，但是这个事情的尺度就要看公司的价值观文化和一些其他的限制了，以及这个员工自己的底线。

就有个段子嘛，就是说去医院看医生。医生说这个病要治5000块钱。病人说不够只有3000块。那医生说，那你就先看3000块钱的病吧。所以说对于管控不好的公司很有可能就是看你有多少钱，你就。

不会香多贵的亲了。那所以说去咨询的时候啊，尽量穿的大方得体，但是别露什么明显logo的衣服，不要让人轻易的推测出你的收入。然后你再透露收入之前，尽量详尽的去问对方服务价格的分档，以及具体的内容。

一定是具体到可以量化的内容，而不是一些模糊的概念，例如销售给你说啊价位更高会给你更优质的资源，这个就是很模糊的那多优质才是优质呢。啊，如果说年收入50万的资源，那这个就是一个明确的标准。如果说有房有车。

那这也是明确的标准。虽然也有男客户会谎报自己的资料给机构啊，但是有一个具体标准之后，你是可以去找他理论的服务的收费高，还有一个可能性是你提出的要求，超出了你婚恋市场价值太多，或者是对应需求的人太少。

那具体计算婚恋市场价值的方法，我会放在下一章。魅力打造片来讲，那我这里先举个例子吧，要求太高，其实就是像早年的网红凤姐。那我们要明确一个态度啊，婚恋市场就是市场，那人也只是各种条件累积在一起的商品。

大家就是看条件。如果你选择了进入市场，你就要遵守市场的定价。那定了超额的定价，那可能就是要给销售回扣。那也就是你定了超出市场价值的标准，你就要多给中介钱啊，现实就是这样子的一套逻辑。

而所谓的对应需求的人太少，意思其实是有钱的凤姐。那例如长得丑，但是很有钱。那想找一个帅小伙，理论上是可以互换的但是平台上觉得哎阿姨我不想努力了。那这样子的小伙子实在是太少了啊。

我说的这个当然只是一个极端的例子啊，但确实也有很多人在相亲平台上提的要求，非常的诡异，像是有的人要特定的某几个星座或者生日需要在上半年不能是下半年，这些无形的都增加了很多没有必要的阻碍。

那这里的话我也要给相亲机构的这些服务红娘去叫声苦，确实有很多客人提这样子的奇葩要求。那对应的奇葩要求就会有更高的价格。那关于婚戒中心的第二个我要和你说的，你必须要知道的事情是在介绍过程中。

他们会运用的一个小套路。那先声明一下，据我所知是没有婚托这个事的。也就是说你见到的每一个人他都是有择偶一。样的那至少在红娘掌握的信心里是有的。至于具体到个人，对方的想法是什么，咱们控制不了。

那但是呢在安排一对相亲的时候，那遇到前面那种我们讲的要求比较超标的客户。那毕竟是收了很多钱，所以需要去保证金主满意，这是婚介机构服务很重要的一个事情。那因此呢，虽然每一次见面，两个人都是客户。

但实际上仅仅针对这一次见面来说，只有一个人是客户，而另一方就是产品，也就是安排一个好的和一个条件差的人去见面，这样子至少能保证其中一个人是满意的。然后红娘再去安抚另一个人的心情就可以了。如果实在不接受。

他就会不计算那一个人的服务次数，然后承诺他说下一次会给他介绍一个更好的。在这里我更要为红娘说一句话来啊，这个也确实是无奈之举。因为即使你没有相亲过，你应该在身边的生活中也听过无数次关于相亲的吐槽了。

基本上相亲就没有几个是满意的，就都觉得对方配不上自己。所以说按照旁观者的标准去匹配投诉率通常会很高。那但是我也声明啊，就这种情况肯定也是少部分确实难以平衡的时候所用的方法。如果说每次都这么用的话。

红娘自己的工作无形也就多了一倍。所以这个明显也不合理嘛。我只是说有这样一种现象，大家需要注意。那说完这两个小的坑，那你究竟适不适合去相亲机构去找对象呢？我觉得有这么几个点。第一种是你有钱没时间。

不一定是很有钱很富裕啊。但是相比较时间来说，你更想花钱解决。那虽然我前面的课程里边讲过社交方法呀，但是会做，不一定代表要做，就好像我会做饭，和我点外卖都能吃饱，而很多人会选择外卖。

他不一定代表我不会做饭啊，我只是花钱去选择一个省事的方式。那第二呢是你心态很好，并且能正视自己。那再说一次啊，婚介就是把你当商品的地方，你如果不能接受被人挑三拣四就不要去。

其实大部分人都是不太能够正确的认识自己的，然后觉得身边的人不识货，平台更靠谱。但是我说过，换游泳池解决不了，不会游泳的问题啊。第三，你对爱情啊激情这些没有特别大的执念，更多的是想找一个相互匹配。

然后生活过日子，细水长流，日久生情。如果说你满足以上三点，那去选择一家口碑比较好的机构，然后放心的去吧。
