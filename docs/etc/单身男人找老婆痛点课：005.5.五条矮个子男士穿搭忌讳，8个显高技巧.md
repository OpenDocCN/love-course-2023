# 单身男人找老婆痛点课：005.5.五条矮个子男士穿搭忌讳，8个显高技巧

矮个子的兄弟如何穿搭才能显高呢？今天这节课呢以我为例来讲两个方面。那我自己呢身高170，但是通过穿搭之后呢，能达到175甚至177的感觉。那今天我主要讲两个方面，第一个呢是穿搭的禁忌。第二是穿搭的技巧。

如果你认真的听讲，并且去执行你也能达到像我一样的效果。那首先第一个禁忌呢是不要穿太宽松的衣服。有的人会说我穿的嘻哈一点，什么吊裆裤啊之类的，上衣也宽松，裤子也宽松，从远处看，就像一个灯笼一样。

甚至说的不好听一点，像矮冬瓜一样。那第二个呢是不要穿七分裤，有的人会去说应该穿七分裤不就显得个子高吗？不是的，如果你穿七分裤，下边又穿袜子，又穿鞋子，从远处看，你的腿是一截两截三截的。

第三个呢是不要穿横条纹的衣服。因为我们的个子如果本身矮的话，你穿横条纹的衣服，在视觉上面会让你这个人看起来更矮。尤其是你在冬天会穿毛衣的时候，一些这个比较宽的横条纹的，从远处看起来真的是很压个子的。

第四个呢是不要穿低裆裤，掉裆裤。这种的裤子会显得你的腿更短。第五条穿搭禁忌呢是穿裤子的时候，不要裤子太长，全部把这个裤脚都堆到鞋面上边，看上去更显腿短了。做完穿搭的禁忌呢，我来讲8个穿搭显高的技巧。

第一个呢是我们要合身的穿搭，上半身的穿搭呢可以尝试夹克。比如说我170，我平常会穿M号的，不要穿到L号，可能外边穿一个M号的夹克，然后里边的。这个衣服下摆会露出来，但是没有关系，这样穿也会有些层次感。

那第二个呢就是我们可以尝试在天气热的时候穿polo衫，也要合身一点的。我们要去穿这种的修身一些的裤子。第二条穿搭显高技巧呢，是穿竖条纹的衣服。但视觉上面也会去显高。

尤其是在夏天我们穿这个衬衣的时候穿一些比较细的竖条纹的衣服，在视觉上面也能达到很好的效果。第三个呢是裤子和鞋子的颜色尽量是一致的。比如说你穿了一个黑裤子，又穿了一个黑鞋，从远处看的话。

会显得你的腿是长的，不要上身一个颜色，裤子，一个颜色，鞋子一个颜色，那看起来就比较短。第四个穿搭显高的技巧，就是用内增高，不要把你的鞋底整得很高，有时候可以穿一些鞋底看起来不是那么厚。

但是里边有很大的这个空间的，我们垫一个两公分3公分的内增高，可能刚开始垫的脚不舒服，但是差不多走个几天也就适应了。那第五个呢是用。发型增高。你比如说我现在的发型，如果我洗完头之后，我给他去吹一吹啊。

多吹高个两公分，3公分也是可以的。头发多少个两公分。鞋子呢多个两三公分，那这就从170变到175了。第六个呢是在我们穿搭的时候，比如说我们穿的上衣，我们可以把这个胸线给提高。比如说有一些衣服它的标志。

如果这个衣服的标志在口袋的地方，那别人看你的时候，人家的视线更容易被你口袋这个位置给吸引。所以我们在穿的时候，有的衣服的标志是在这个位置，胸线以上的那对方在看你的时候。

他的视觉就不自主的停留在你的高处这个位置。那第七个呢是腰线，我们要去拉高一点。你比如说我们上边穿了一个马甲。那我们下边在穿裤子的时候，我们就把裤子稍微往上去提一提，穿那些修身一些直筒一点的裤子。

但是不要紧身裤啊，紧身裤穿上去很像精神小哥。第八条呢是跟您接触的时候，要经常的挺胸抬头，对吧？你不能就是。这样的一弯腰啊，然后脖子再一垂下来啊，你本身可能170的就看着像167了。
