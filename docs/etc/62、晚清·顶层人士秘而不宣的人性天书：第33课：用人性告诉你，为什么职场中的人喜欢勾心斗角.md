# 62、晚清·顶层人士秘而不宣的人性天书：第33课：用人性告诉你，为什么职场中的人喜欢勾心斗角

亲爱的小伙伴们，今天我们要分享人性职场篇的第二课，用人性告诉你，为什么职场中的人喜欢勾心斗角。相信每个刚进入职场的小白们都没有要勾心斗角的想法，大多都是抱着把本次工作做好，和同事们友好相处。

开开心心的工作的想法。就像很多姑娘刚嫁入婆家时，一心只想着好好对待公公婆婆，希望公婆也能善待自己，和婆家的所有人都和谐相处。大家就像真正的一家人一样，根本不会想着要和婆婆针锋相对。

但结果呢大多数人进入职场不到两年，就会发现，哪怕你再不想勾心斗角，职场依然处处充满了勾心斗角，没有点心眼，要么永远是炮灰，要么和升职加薪无缘。前几天有位学员和我说，他换了三家公司了，想找一个氛围好一点。

简单一点的公司。但是最后发现每个公司都充满了勾心斗角。为什么人与人之间不能好好相处呢？我一为年纪，他说今年2222岁有这种想法也可以理解，毕竟年轻嘛。但假如你32岁，42岁，还有这么天真的想法。

那就不容乐观了。作为一个成熟的人，你得明白，职场中的勾心斗角就是一个常态。平常心理解就行。并非大家一定要勾心斗角，这是生存法则决定的，每个人都需要维护自己的利益。你的职位越高，勾心斗角的现象就越严重。

大部分情况下，勾心斗角不是本性而是利益驱使。接下来我们就从人性的角度来分析一下职场中勾心斗角的内在逻辑。现然我们经常分析的是职场，但一通百通。其实在其他场合也是一样的。第一个原因，历史文化背景使然。

中华民族有着5000年悠久历史，在整个人类发展史上，斗争几乎贯穿了整个历史。我们从秦始皇统一六国开始讲起，秦始皇死后，最有可能继位的就是公子扶苏和胡亥。据传，秦始皇驾崩前曾留下诏书，传位于公子扶苏。

但李斯联合赵高篡改了赵书，把胡亥扶上了皇位，最终导致大秦帝国二世而亡，这么一个庞大的帝国只存续了二十几年。李四赵高为什么要帮胡亥继位呢？因为当时扶苏更得人心，是儒家代表，如果扶苏继位。

这两个人可能就无法维持之前的荣光。但如果把胡亥扶上去，就有了从容之功，才能权青草野。曹丕有很多儿子，但最喜欢的是曹昂和曹植。曹昂早逝，但最后曹操的权柄不是曹植继承，而是曹丕继承。曹丕当了皇帝后。

几度迫害曹植七部城市的典故就是这么来的。唐太宗玄武门之变，最终登上大唐第二任君主的位置，明朝朱棣抢了侄子朱允炆的皇位，康熙朝史九子夺嫡，无一不是勾心斗角的巅峰啊。因为皇位只有一个啊，人人都想当皇帝。

不去勾心斗角，没有自己的份啊。在女人的历史上，勾心斗角就更多了。一部以后宫嫔妃斗争为主题的电视剧甄嬛传红到发紫，哪怕时隔十年都还在不断重播。这部剧为什么这么火啊？因为处处充满了勾心斗角。

可谓是宫斗剧的巅峰。而她的火热也说明女人就是喜欢看这些斗来斗去的剧情。这些女人为什么要斗啊？为了皇帝的宠爱和家族的荣耀。皇帝就一个人，他的宠爱就这么多，给了别人就不可能给你了。生多粥少，不去尊从的话。

根本轮不到自己呀。有了皇帝的宠爱，在后宫里吃穿可以更好，做人可以更有底气，还可以让自己的家族更加辉煌。有这么多好处能不去斗吗？所以一群女人逗得不亦乐乎，一旦开始宫斗，那就骑虎难下了。

因为输的人不但输了龙宠，可能连性命都不保了，能不全力以赴去斗吗？这就是我们的历史文化背景呀。可以说没有哪个民族有我们这么悠久的斗争历史。连一位伟人都说过，与人斗其乐无穷。

说明我们骨子里本来就有斗争的因子在这是其他国家无法比拟的。就像在商场里一旦开启手段，外国人基本不是我们的对手，他们哪里斗得过我们呀。延伸到职场里，虽然职场比朝代小很多，但道理是一样的。

有人的地方就有是非就有斗争。第二个原因，利益驱使。如果没有利益的时候，你就算让大家一起勾心斗角，可能很多人都没有动力，毕竟斗了有好处吗？没有好处的时候，谁愿意操这个心呀？但只要利益驱使。

大家就会斗得不亦乐乎。职场就像一个金字塔，大家一起向金字塔走去，越走到上面，位置和空间就越少。如果你不把别人挤下去，自己就会掉下去。所以下面的人想努力往上爬，上面的人一心想要守护好自己的位置。

还要想办法继续爬得更高。对应到一个公司里，就是不可能人人都是领导，不可能每个人的薪水待遇一样。主管经理总监的岗位是有限的，谁做上去了，就意味着可以领导别人拿更多的薪水。这种吸引力有几个人拒绝得了？

这也就是蛋糕原理，现在有3块蛋糕，但很饿的，很想吃的人有300个不吃就只能继续挨饿，谁都会打破脑袋去增抢这3块蛋糕。可能会有人天真的提出来，为什么不把这些蛋糕切成很多差不多的小块呢？这样人人都有的吃。

也不会抢了。在现实生活当中，这几乎是不会实现的，或者说哪怕真这么做了，就人人满意了吗？不会，因为一个公司中有贡献大的人，也有贡献小的人，如果人人都一样，那么对贡献大的人就不公平啊，所以绝对的公平。

其实是最大的不公平，而且会导致所有人都没有奋斗的动力，最终形成自己不想努力，就指望别人努力的情况。最后别说蛋糕了屁都没得吃。第三个原因，社会规则的运行逻辑使然，我们想想看，我们活着的目标是什么？

难道就是混吃等死吗？不，我们有很多需求。首先，我们希望自己能够养活自己，让自己活下去。这就是我们为什么要去工作啊，养家糊口才是工作的底层逻辑。什么为社会做贡献啊，为谁服务啊。

那也要建立在自己的生存问题得到的解决以后，要是点这些问题都没有解决，你会有这么高尚的追求吗？当我们能够生存下去的时候，我们又会想着如何让自己活得更好。比如每顿饭都有肉吃，能买漂亮的衣服。

能够住上宽敞的房子，这就是我们对生活的进一步要求。当这一切我们都实现了以后，我们又有新的要求了。我们希望得到别人的尊重，希望出门的时候能够被别人高看一眼，这已经是精神层次的要求了。

还要实现这些的关键就是你的事业成就。谁的成就更好，谁就能活得更好，不但可以让自己活得更好，也能够让自己的家人活得更好。这也就注定了我们活在这个世界上必须去竞争。在这里也和大家再聊聊人性的底层逻辑。

用我们对外的第一套规则，我们讲究的都是我们要和谐，我们要团结，我们要友爱，是不是？但事实上，任何社会真正实行的是第二套规则，丛林法则，弱肉强食，适者生存，没有人会因为同情你而给你更多蛋糕。

你想要更多蛋糕，必须自己去努力，哪怕是去乞讨，你也要乞讨出心意来，打败其他的乞丐，就是这么简单粗暴。但为什么要提倡第一套规则呢？因为要保护羊啊，羊都活不下去了。请问狼吃什么，把羊都吃完了。

狼迟早也会全部饿死。所以狼们都明白，必须要有源源不断的羊，足够数量的羊，才能保证自己的生存。放在羊身上也是一样的，羊要活下去，就必须吃草。但如果把草根都啃完了以后就没有草可以吃了。

所以应该这里吃一会儿就换个地方给这个地方重新长草的时间之后才能够回过头来继续吃。这就是食物链的真相啊，谁要是不遵守这个规则，又或者是哪个平衡被打破了，那么就会面临灭绝。动物的世界如此。

人类的世界也是如此。职场不是与世无争、无忧无虑，与现实社会完全隔绝的世外桃源。既然大家都想让自己的利益最大化，就必然需要增强，就会出现人与人之间斗来斗去的现象。人与人之间的权力争斗、勾心斗角。

就是职场上的竞争行为。你厌恶这些竞争行为，说明你还不能接受竞争的普遍化以及常态化，就暴露了你的天真和无知。如果你不敢和别人去竞争，只会一味的谴责和逃避，只能说明你知道自己无脑。第四。

领导希望你们存在一定斗争。我们在上一节课里分享过，领导不希望所有人都万众一心，这样他都会担心自己被架空。其实严格来说，领导内心也是很矛盾的，他不希望下面的人特别齐心，随时会对自己产生威胁。

但他同样也不希望下面的人斗得你死我活，或者大家都忙着勾心斗角，没人干活了。这也是领导不愿意看到的。所以他最希望的是能够保持在一个很合适的度。那么大家为了增强最大的资源，势必会主动来和他交心，当他的心腹。

这样他才会有安全感。另一个原因，作为领导必然注重业绩，如果大家都很佛系，业绩怎么可能做得出来呢？想必大家都知道鲶鱼效应吧，挪威人靠海而生，喜欢吃沙丁鱼，尤其是活的沙丁鱼。但是沙丁鱼非常娇贵。

当渔民把刚捕捞上来的沙丁鱼带回港口后，大部分沙丁鱼，都会因为适应不了离开大海后的环境和窒息而死。而死了的沙丁鱼味道和口感都会变差，价格比活鱼低很多，而且还很难卖得出去。为了获得更高的利润。

渔民们想尽一切办法让沙丁鱼活着到达港口。最后有一种方法最有效。渔民在装沙丁鱼的鱼草里放入了几只沙丁鱼的天底。鲶鱼。沙丁鱼为了活命，就会加速流动，四处躲避，从而保持了旺盛的生命力。

沙丁鱼缺氧的问题也得到了解决。因此，加入鲶鱼后，大多数沙丁鱼都活蹦乱跳的到达了港口。这种被对手激活的现象在管理学上被称为鲶鱼效应。作为领导，哪有不懂这个管理现象的呢？

所以我们要一分为二的看待职场中的勾心斗角，在一定程度上，大家在勾心斗角，说明还都有上进心。如果人人都直接躺平了，每个人都和和气气的，对一个企业或者部门而言，绝对算不上好事。

以上我们讲的大多数是因为利益而陷入了斗争。其实还有一个原因，也就是第五个原因，看不惯某人或者看不起某人。大家看过如懿传吧。有一个剧情是这样的，海澜希望能够分配到如懿的宫里和如懿一起住。

结果皇后和高曦月不想成全她，就把她分到了高熙月那里。当时高希月是贵妃，海澜只是一个常债，两人的级别相差了十万八千里。加上海澜的性格，懦弱胆小，怎么也不可能故意去惹高曦悦。但高熙月就是不想让海澜好过。

先是打发他去最差的地方住，甚至连体面的宫女都比她做的好，还诬陷她偷了红罗炭，让她跪在雪地里，狠狠的羞辱她。海南的存在怎么也不可能威胁到高息月，也可以说完全影响不到他的利益。那么，高息月为什么要诬陷他呢？

一是讨厌他每天如一长如一短的。二是对海澜那种怯怯懦诺的样子，很看不上眼，所以就时常去欺负他，给他难堪，羞辱他。在职场中也是一样的。有的时候未必一定要有利益冲突，有时候就是纯粹讨厌一个人。

比如有的人因为家里穷，在职场里的存在感就很低很自卑的样子。可能就会有人忍不住去欺负他，太弱的人，有时候就会勾起别人人心中的恶，会觉得不来欺负你一下，都觉得对不起你。

所以在职场里千万别让自己表现的像个大丫鬟，谁都可以拆遣你谁都可以对你呼呼呵喝。我们不是说一定要活的嚣张跋扈，让别人不敢小觑，而是要活出自己的底气，让别人轻易不敢来犯。既然我们知道了勾心斗角的底层逻辑。

接下来和大家分享几条面对勾心斗角时的正确做法。第一，任何地方的人素质都是参差不齐的，所以平常心对待，不要觉得勾心斗角是多么了不得的大问题。有人的地方就有斗争，不要大惊小怪，影响自己的工作情绪。

到时候勾心斗角没有斗到你，你自己的情绪，把自己害死。第二，再怎么斗争，别影响公司利益。在古代，两个大臣怎么斗，怎么吵，皇帝只会当何师佬。但如果你敢通敌卖股试试看，马上诛你九族。所以。

任何斗争中不伤害公司利益是原则，否则你斗争水平再高，最后也是炮灰。要是你为了陷害别人，把机密透露给对手公司看看，除非不发现，一旦发现，身败名裂。第三，守住底线，有进有退，想要更大的利益。

其实没什么好诟病的，谁不希望自己过得更好呢？但再怎么竞争都不能失去做人的底线。你可以把对方做错的事儿告诉老板，但你不能捏造他做了什么错事儿。就像古代的强盗，虽然靠抢劫为生，但是他不抢老弱病残。

这就是道义有道。真的自己想要的，但是不失去自己做人的底线，这才是真正的赢家。这节课我们就分享到这里，希望大家对这些现象有更深刻的理解，也能够更好的在职场上保护自己。
