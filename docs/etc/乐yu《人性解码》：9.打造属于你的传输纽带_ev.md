# 乐yu《人性解码》：9.打造属于你的传输纽带_ev

。在这节课里，我会带着大家一起看看沟通中最重要的一些原则。你有没有见过一些情侣或者夫妻吵架。哎，吵着吵着突然就变成了互相的人身攻击，再上升到一些完全没有关系的高度。比方说老公因为在外面应酬。

有段时间很晚都没有回家。老婆忍了一段时间在某天终于忍不住了，就开始跟老公抱怨说，你每天去那么晚，你就不知道早一点回家的吗？老公说自己在外面有事忙，忙就不能早点回来吗？能每天都那么忙吗？哎。

老公听了就很不爽了。哎，我每天在外面辛苦应酬，你还在这跟我闹，你烦不烦，老婆一听就更来气了，你这什么态度啊，你对这个家还有没有责任感的哦，然后就上升到老公肯定在外面有人的，否则就不会这么晚回家。

老公肯定是变心了。像这个例子一样，如果你违背了初心，不专注在一开始的目的交流。沟通很容易会走偏，因为沟通的本质非常的简单，它只是一个信息的交换和传达，并不是针对错真输赢，尤其是在亲密关系上。

像上面的例子。如果老婆一开始就跟老公说，哎呀，你这段时间回家比较晚，我有点担心，我也想你可以早点回家陪陪我。这个才是老婆原本想表达的内容。再时老公听完就不会有任何毛躁的情绪，更多是一种理解。

甚至带着愧疚的感觉。其实沟通是多样性的语言上的沟通，只是其中之一。之前咱们也学过越负面沟通，越会产生一个负面的回应。越是正向沟通，也越会有一个正向的回应。这就是正循环和不循环的产生。那正面的沟通呢。

是自愿平等的。就是你可以说你想说的，我也可以说我想说的，当你不想说的时候呢，也可以不。说但很多人就会强迫对方，哎，你有什么你就说出来嘛，所以截停负面沟通，进入正向循环很重要。当对方表现出啊。

或者明确告诉你，哎呀，不想跟你聊了，那正循环的开始一定是遵循平等自愿的原则。对方想停止不聊了，你只需要回应好的就足够了。因为有能力说出好的，并且不继续追问的人，意味着他有很高的认知能力，知道该干什么。

不该干什么。有很强的自控力不纠缠，有更好的办法来处理当下这种负面的沟通，真正高质量的沟通啊，一定是专注不分神。为什么很多人说哎总跟不上对方或把对话搞砸了呢？因为很大一部分原因是因为你跟对方沟通的时候。

总想着要怎么回应，或者想着其他的事儿，就等于开车一样，如果你一边开车一边吃东西，你还一边打电话，那发生车祸的这个概率就会很高。当你脑袋里想着哎呀，我要说一些什么呢？你就没办法去。今经人家说了一些什么。

因为在沟通里，我们总想表达自己的观点和想法，并且希望对方能认同每个人都这么想。其实对方也是这么想的。如果你能让他觉得哎，你是很懂他，你很理解他，那你们的沟通质量就会非常的好。

能够真正的理解对方分为三个方面。第一，你能不能听完人家说的话，因为很多人习惯性的强话，所以能听完不仅是一种礼貌，更是一种能耐。那第二，你是否能听懂别人的话里前台词，他真正想表达什么。理解不到。

就不可能在同一个沟通层面，更高层次是你是否能读懂人家表达的细微情绪。在这里呢，我再次强调真正的沟通都是在情绪上的沟通。所以我给大家总结了很好记得一个沟通方法，三进两出。进指的是你主动接收的信息啊。

比方说第一个对方在一个什么情绪状态里头。第二件是他想表达什么内容。第三件事这个事实是怎么样的？比方不同的语气状态，同样的内容，他也有着天渊之别。比如说我没事儿跟我没事儿。传达出来的情绪是不是完全不一样？

还有你不喜欢吃就别吃了。哎，对方在生气的时候和调皮的时候，你的回应也应该是不一样的。还是那句话读懂情绪状态。那出是指什么呢？出是指你要表达的信息，第一，你这么说，对方会有什么想法。你得换位思考。

你能说到他想听的，那结果自然就会往你想要的方向走。第二，你希望他回应你什么？也就是说用对方喜欢的方式来达到你想要的结果。比方对方是个很直爽的人，你跟他沟通的时候兜兜转转，他肯定是不喜欢的。

或者你遇到了一个比较含蓄的人，你一上来单刀直入，哎，就让他很不爽了，所以不要关注你想表达什么。关注怎么样才能拿到你想要的结果，其实在我们日常交流中啊，很多话都是没有意义，没有用的。

你只要捕捉到其中的这个关键词就可以。展开很多的话题和深入沟通。你只需要沉下心来倾听就可以了。比方说我今天挺想吃披萨的。哎，我昨天回家路上捡到了一只很可爱的小狗，或者回来的路上买一些水果吧。

你看是不是很容易就能听出这些重点在哪。而且我们往往都把重点放在后面，就是重点后置，所以觉得自己不懂聊天的人了，你只要抓住关键词发散，根据自己对这个词的感受，会有过去的经验去回应。

那就可以有源源不断的话题，比方刚刚说的回来的时候买些水果，关键词很明显就是水果，对吧？多做这样的发散练习，慢慢你就会觉得聊天其实真的非常的简单。在沟通当中啊，很多矛盾都产生。

是因为在你自己的想象里给事实添由添处，就像一开始的例子，如果你的伴侣回家绷着脸，不怎么说话。你问他他也有一搭没一搭的，哎，你觉得是他不想跟你说话，所以才绷着脸的。然后你就在自己的脑海里各种的脑补。

但真相不一定就是你以为的那样，所以不要先下定论，不求感觉，但求事实，因为在我们的脑海里的想象画面了，多数都是负面的。你的猜忌很容易让你产生情绪，产生情绪就很容易伤害你自己，去接近真相，了解真相。

方法也非常的简单。在你问为什么之前先确认是不是。比方你觉得对方不想跟你说话，你不要一上来就问，哎，你为什么不想跟我说话呢？取而代之的是，你可以试着问你是不是不想跟我说话了。

这个就是先确认你可能会发现你认为的，不一定就是你认为的。沟通的时候啊也可以尝试多点的表达。我不是很明白你的意思。你刚是不是想说这个这个这个或者少一点误解，少一点偏见，让事实回归事实。

让沟通回归到信息的交流和分享，而不是情绪的对抗。🎼。
