# 单身男人找老婆痛点课：007.7.五条胖男士穿搭禁忌，7个穿搭显瘦技巧

今天这节课我来讲一下身材比较胖的兄弟，在穿搭的时候，一些禁忌以及如何穿搭才显得自己比较瘦。首先是穿搭禁忌。第一个呢是在穿衣服的时候，尽量少穿横条纹的衣服。

因为横条纹的衣服在视觉上边会让人看到你的身材是延伸的那会显得更胖。呃，第二条呢是在穿衣服的时候，尽量少穿亮色系的。比如说白色的，或者说一些黄色的。这些在视觉上边，你本身比如只有150斤。

但是穿上去之后可能显得有160斤，170斤甚至180斤。第三个呢是在选择衣服质地的时候，不要选择那些布料很差的，没什么垂感的。尤其是夏天有一些这种的T恤，它比较薄，它可能也是牌子货，但是穿在身上之后呢。

就显得你走一走，风一吹，就显得你的肚腩了，得一些布料稍微厚实一些的，垂感质地强一点的。第四条呢是尽量不要穿紧身裤，尤其是有一些兄弟，他的腿比较粗。如果你穿的裤子太紧身，你一走你的整个腿型就露出来了。

可以选择一些垂感比较强的直筒裤。那第五个穿搭竞技师不要穿低帮鞋啊。尤其是如果你穿了一个七分裤，九分裤穿着袜子，穿着低帮鞋，就会去显得你的脚脖很粗，如何穿才能让你显得更瘦呢？分享几个技巧。

第一个呢是穿竖条纹的衣服，竖条纹的是纵向拉伸的，看起来你的身材也会比较瘦一点。那第二个呢是在选择衣服，布料的时候，选那些垂感强的。同样一件T恤有180克的，有230克2吧60克的那你就选择克数更重的。

第三条呢是要避开亮色调的衣服，去选择一些冷色系的衣服。比如说黑色呀，选择一些蓝色呀，四条是可以选择一些牛仔的夹克。因为如果我们单独穿一个显得自己胖了时候，我们外面穿一个夹克也能起到修饰的作用。

或者穿一些牛津纺布料的衣服，第五条呢不要穿低帮的鞋子。改成穿中帮或者是高帮的靴子，躺在你的脚窝。如果穿的好，能够看起来你的腿也是比较长的那第六条呢是选择直筒裤，选择那些修饰腿型的裤子。

不要选择那些低档的紧身的裤子。当然也是靠你多去衣服店里边多去挑一挑，甚至让人家卖衣服的，帮你去搭配一下。第七条呢，我们在穿两件的时候，比如说里边一件，外边一件，我们的内搭尽量是穿成黑颜色的。
