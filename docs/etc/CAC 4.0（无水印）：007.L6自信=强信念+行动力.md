# CAC 4.0（无水印）：007.L6自信=强信念+行动力

好。说完了所有的让你不自信的东西，我们接下来要说的是如何获得真实的自信。首先我给自信下一个定义啊。我觉得用这个定义是方便我们解释很多问题。心理学里边本身是对自信这个事是没没有一个定义的。

因为不是一个心理学概念，可能有相近的是自尊，可能是信心、自我效能感之类的。但我所以我这里给一个定义啊。就是你始终会认为自己是有机会的，并且会去采取行动。第一个就是你有一个正面的感觉，认为自己是有机会的。

就像我说的，我他妈突然腿断了，我腿断了以后，我认为我依然那怎么办呢？我在现在基础上，我再继续去努力。也许30年以后赚了很多钱，那个时候有牛逼的假肢功能，脑控假肢，然后就安上了。

所以更第二点更重要是你在采取行动，不是他妈的自己在家里意淫。😡，就是对吧就是有一些人比如学情感，他就操觉的就是理论派专家，完了根本不敢采取任何行动。那什么是不自信？其实就是反过来。

是在负偏负面的情绪和视角在主导你。同时你的行动被阻碍了。那行动被阻碍了。这个大家只要明白这个概念就行，你也不需要让这件事情去定义你自己，你不需要让他去定义你自己拿他去评价我我自己。

但你要客观的明白这个概念。那么除了除了私信以外，刚刚s在是提到的自恋，什么是自恋呢？之恋觉得我他妈不可能失败。😡，我怎么能不完美呢？我怎么能失败呢？或者事情怎么能不完美呢？这就是自恋。

比如说如果我们去做线下的时候，一开始你做不到完美，然后你觉得我怎么能做不到呢？你就开始自我攻击。或者别人给别人给你的坏反应，让你破防，不是因为他不认可你让你破房，而是因为我居然没有能一次就学了这么多了。

我居然没能做到完美。让你觉得破防。嗯。或者有时候我们会对别人的要求也是会完美的。因为自恋的意思是说，我对全世界我是宇宙的中心，我对全世界任何人和任何事都有掌控，任何事情不受我控制的时候都会戳到我的自恋。

让我意识到，我他妈不是宇宙中心的主宰。😡，那这个时候我就不舒服了，所以自恋的人会出现拖延、借口、退缩之类的这些东西。
