# 白鸭《核心魅力炼成-训练课》：01.第00集 前言

Yeah。大家好，我是白鸭。一个帮助过上千名学员提升个人魅力的咨询师。🎼十几年前，我在国外生活过一段时间，那个时候因为乐观外放的性格，加之从小就很喜欢体育运动。我很快和当地的小伙伴玩成了一片。

跟大家都成了很熟悉的朋友。当时呢有一个外国朋友跟我说过一句话，让我至今印象深刻。他说感觉你一点不像你们国家的人，更像是我们国家的人。我很奇怪的问他，我怎么不像我们国家的人了。😮，他说你特别自信。

🎼我认识的你们国家的人，没有你这么自信的。🎼我知道这个朋友他讲这话没有恶意，但当时听完我还是觉得很诧异。后来我开始留心观察身边的人，我发现亚洲国家的人，我们在与人打交道的时候。

相比西方国家的同龄人确实普遍缺乏能量。给人感觉唯唯诺诺的，当众的时候不敢讲话，做游戏的时候放不开，在感情里动不动就被牵着鼻子走。说通俗一点就是缺乏人格魅力，不够吸引人。🎼后来我发现这个问题它很大。

解决它很有意义。于是我就开始了关于生命能量方面的学习和研究，阅读了大量的书籍，包括心理学、社会学、人类学，还有哲学。🎼我发现自己身上有很多有能量的特质，比如从来不会认为选择有对错之分，遇到任何问题啊。

都保持乐观积极的态度。生活出现不如意，也会享受当下。对世界永远充满好奇，愿意在人生体验上投资等等。🎼我开始把这些和我身边的人分享，也组织了很多关于个人成长方面的活动。后来我开始从事咨询行业。

十0年来帮助过数千人，从负能量体制转变为积极乐观的人。🎼从不自信变为自信。也帮助很多人从不幸福的人际关系和感情泥团里走出来，重新把握自己的人生幸福。我把这套方法总结成一套建立强大内核的系统课。

它可以帮助你散发无限的个人魅力。这个系统课一共十二节，第一节第二节是理论课。第三到第十二节课是我十1年来总结下来的训练方法，配合实际发生的经典案例。🎼只要你跟着我的思路学习完这套方法进行内核训练。

不管你现在是深陷感情纠葛，还是遇到了人生的迷茫，训练完你都会发现你自己是一个魅力无限，有着强大内核力量的人。
