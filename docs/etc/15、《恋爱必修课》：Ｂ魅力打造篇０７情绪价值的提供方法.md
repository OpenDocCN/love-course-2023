# 15、《恋爱必修课》：Ｂ魅力打造篇０７情绪价值的提供方法

你好，我是俊瑶，欢迎收听俊瑶的恋爱必修课。啊，今天呢我们来说一个，但凡是听过我讲课的同学，一定会非常关注的点啊。情绪价值。我一直说情绪价值是大家逆袭的一个重点。那究竟什么才叫做情绪价值，以及怎么提供呢？

今天我就来详细的说明一下。情绪价值其实很容易和情商混在一起啊，但实际上还是有挺大的区别的那我们经常看到一些明星高情商的片段，那有些呢是在尴尬的场合去原场。

那我最近看到比较有印象的是前段时间这个有一个综艺节目叫做我是演员啊，第三里边呃于正那很毒舌的去批评了江潮，说他的演技太差了，还迷之自信，那个场面一度尴尬。然后江潮呢回复说于老师啊。

以前呢只是一部分看过我戏的人知道我演技差，现在全国人民都知道我演技差了。那这样子尴尬的场面一下子。就缓解了，就单纯说这个片段来说，其实他没有提供情绪价值，但他是一个情商高的片段。那我再说一个画面啊。

就有一个人是情商高的代名词黄渤。那比较经典的一幕就是他去上那个鲁豫有约的节目的时候，鲁豫问他，你现在觉得自己特别火是吧？那鲁这个黄渤就回答鲁豫说，哎呀，我都来鲁豫有约了，能不火吗？啊。

那后来过了几年又有一次也是上鲁豫有约。鲁豫说哎，你可以稍微做的近一点。然后黄渤说，哎呀，我这个没有，我平时习惯了，见到美女就紧张。那其实这几个这两句话都是很典型的提供情绪价值。也就是抬高对方。

然后说漂亮话，让人觉得开心。那又是高情商，但他其实提供了情绪价值。那黄渤呢还有一个非常经典的场面啊，是在2013年的金马奖的颁奖典礼上。

黄渤呢作为颁奖嘉宾去被郑玉玲和蔡康永就相继的去刁难这个场面啊几度尴尬。那最后都被完美的化解了，甚至很婉转的怼了回去。那这个也是现在大家一提到高情商就会讲到的非常经典的一段画面。啊，这里的高情商呢。

其实他是回怼。那其实就完完全全不是提供情绪价值了，完全相反的。所以说总结来看啊，就可以给大家一个比较简单的类比，叫做赚钱和智商。赚到钱是智商高的一种体现，但是赚到钱不一定就是智商高。

而且智商高也不一定就赚到钱。但是智商高的人普遍能赚到更多的钱。大家能理解吗？就同样的啊，能提供情绪价值的是情商高的一种体现，但是提供情绪价值不一定需要高情商，但高情商的人普遍是可以提供更高的情绪价值的。

有一点绕啊，大家可以重新重新的听两遍，然后仔细的理解一下。OK那如果说用简单的方式去定义一下情绪价值的话，它是一种让人情绪愉悦的能力啊，简单来说呢可以分为以下的几个方面。第一呢是消化负面情绪。

就让人开心的第一件事啊，其实是可以帮助一个人去缓解自己的不开心。而我们身边一些亲密的朋友啊，相互之间没有什么利益关系，为什么还能在一起，而且感情越来越深呢？

其实就是彼此花了大量的时间去帮助对方消化负面情绪去开导啊，安抚呀，或者只是简单的陪伴，给你一个拥抱。这个这种情绪价值呢是最基础的，也可以说是每一个人都可以提供的。当然有的人可能更会说话呀，更会安慰人。

但每一个人都能做到的这个基础值很高。那但是呢生活中我们也见到那种人啊，就你明明正在伤心难过的时候，他过来跟你说，哎，你这伤心没意义啊，你要改变，你要努力啊，或者说哎呀这件事有什么值得伤心的呀。

那如果从客观来说的话，可能他说的话是对的。但是他在这个不合适的时候说，我们通常说这个人没有眼眉，那也就是情商低。那但是呢其实他也没有索取情绪价值啊。所以说啊再次说明了情商和情绪价值，他没有。

必然的联系啊，那消化负面情绪啊，还有另一个方面就是能接受对方的坏脾气。那男生呢可能容易暴躁爱嚷嚷啊什么的，然后女孩呢可能会有很多的小脾气啊，任性啊作呀。

这些都是通常别人忍受不了的那如果你可以去比较平静的接纳，而不是和对方正面的去争吵，可以比较化解这种矛盾。那这个将是提供情绪价值的一个重要体现。那情绪情情绪价值的第二个点呢叫做带来正面的情绪。

那最典型的是愉快、积极和新鲜。就好像我们总说这个好看的皮囊千篇一律啊，有趣的灵魂万里挑一。那幽默感在交往的过程中啊，一直是一个很加分的事情。不过强调一下啊，幽默感不一定是你讲笑话。

而是你对幽默的一种感知能力。那如果说就同样一个笑话，他讲出来你能get到点，而别人get不到点的话，那这个人也会对你产生好感。啊，同时我们也总说啊爱笑的女孩运气不会太差。就如果一个人在和你在一起的时候。

他可以感到非常积极，能量充沛的状态，他受到感染，那这个也是一种情绪价值的获取。而新鲜感呢同样也是，那不过是随着相处啊，新鲜感本身就会变淡，而且呢尺度掌握不好，新鲜感容易变成所谓的神秘感。而交往久了。

如果你还保持着神秘感，反而会造成反面的一个影响。别人觉得你隐瞒对自己不真诚。那不过呢在这个初期交往的时候啊，新鲜感还是一个挺不错的情绪价值的。情绪价值的第三点叫做满足虚荣心。

那这一点其实是因为对方啊从而接收到其他人的反馈之后，我们所得到的感受。就好像男生会想着带这个漂亮的女朋友出去聚会嘛，可以炫耀一下。那女生呢会拿男生给自己买的包包啊、首饰啊出去炫耀啊，所以好看。

其实也是你能给对方带来的情绪价值的一种方式。所以说如果在社交场合中，你能特别给另一半长脸，这个是特别加分的事情。而第四点叫做给予存在感。那有一句话叫做我爱你，不是因为你是谁，而是因为和你在一起的时候。

我是谁那所以说就一个人就给人极大情绪，一个给人极大情绪价值的人，他是让人可以重新爱上自己的人。那我们每个人肯定都是更希望自己当明星，让别人追逐这种感觉。但如果你能给予对方存在感。

那你是那种当主持人的角色，然后把聚光灯去让给了别人。然后自己呢去做那种最理解、最支持、最欣赏他的人。那这种会给予对方极大的存在感，那也是非常重要的一种情绪价值。

所以说呢啊尊重理解、认同、支持、共鸣、包容，这些都是在给予存在感的关键词。那而且还有一种说法呀，就叫做当一个女人对一个男人有了崇拜，那就什么都有了。

你会看到很多的情侣在一起给人的感觉就是这个男生是哪这个女生心中的英雄啊，就感觉好像无所不能一样。就在这种状态下，男生通常对女生都爱的不行。所以说呢其实在我们的日常视交中也能感受到啊嘴甜的人特别招人喜欢。

啊很多时候我们都觉得自己自夸特别尴尬。但是如果别人一直这么夸你，即使是夸张的，你也会喜欢听。而且慢慢的你会觉得哎这个人才是懂我的人啊，只有他才识货。那最后我们总结一下今天的内容，情商并不等于情绪价值呃。

情绪价值是一种让人情绪愉快的能力。那其中呢包括消化对方的负面情绪，那带来积极的正面的情绪，那满足对方的虚荣心以及给予存在感四大板块。
