# 15、《恋爱必修课》：Ｂ魅力打造篇０３朋友圈是你的社交橱窗概念篇

你好，我是俊瑶，欢迎收听俊瑶的恋爱必修课。那今天呢我们来聊一聊在这个时代，你最重要的社交利器朋友圈。我现在慢慢发现啊，越来越多的人开始不更新朋友圈了，或者三天可见。但朋友圈其实是你正在努力生活。

并且接受社会化的一个象征。正在努力啊，是指你还是一个接受生活不确定性的正在努力的人。那如果说你已经事业有成，然后家庭美满，年纪也大了，我觉得不更新也就不更新了。但是如果你还希望认识新朋友。

开展新的恋情工作也进一步。那朋友圈是你一个辅助的工具。那对于新认识的人啊，朋友圈就像是你的简历一样，你可以让对方快速的了解你是一个什么样子的人。而什么情况下，你才不需要简历呢？那一个是你不需要在找工作。

也就是上面说的你的生活已经定型，并且你不希望再改变了。那其实啊朋友圈是一个简单高效的社交维护工具。你把自己想成一个品牌。那微信其实就是那最火的一条商业街，你的所有朋友都会来这条街上逛街。

而朋友圈就是你的橱窗，你把你最新最好的内容放在橱窗里，他们就会记住你的存在，时常在橱窗面前看一看，如果有兴趣，他们就会进店逛一逛。也就是过来私聊你。如果你不更新朋友圈，而是总去私聊。

这个就好像是接上去发传单拉客。对方有兴趣可能会进店看一看和你聊几句。如果对方没有兴趣，或者当下没有时间可能就不理你走了，而你的姿态，完全不是一个名牌的姿态，而是一个打折处理的姿态。

那同样还有一个理由告诉你为什么不要朋友圈三天可见。就好像我们上学考试，已经说明了这次考试是开卷考试，你可以带学习资料来考试。但是你硬要说我不用，那无非两种情况嘛。第一，你不想好好考试，成绩对你无所谓。

那第二种呢是你已经全部记下来了，根本没必要带材料，那也能考好。那第一种情况就是不想社交，不想有任何新的社交关系。那第二种就是你太牛了，你可能是明星，你可能是网红你可能是某一个位置的领导。

别人啊都求着认识你，如果说你不是以上的两者，那考试的时候，你就最好带上你的。教材而社交的时候用心更新你的朋友圈，时刻记住，朋友圈是你的社交橱窗。而我要和你说的第二件事。

朋友圈是我们每一个人刻意表达的一面。或许你会说啊你更新着玩，不想那么多心机，就像是当日记一样记录。但是朋友圈有一个非常本质的不同，是点赞和评论功能。当你发一个内容，你微信里哪一群人更爱评论。

哪一个类型的内容点赞更多。那长久以往每一个人都会下意识的去修正自己发布的内容，所以说不要相信任何人他的朋友圈是乱发的。真正发出来的东西，都是他们想要表达的东西，那只能说他们发的够不够专业。

那可能有的人和你说啊，自己是乱发的，甚至还说哎你发的这么用心精致，太有套路了，太刻意了。我是建议大家不用太在意这些说辞。有一句话一直鼓励着我呀，我也分享给你。成长的路上没有什么真心的鲜花和掌声。

收获更多的是弱者的愤怒。这些人大部分人其实都是做不到你这样子认真努力，但是又不愿意承认自己懒惰无能，所以经常会说哎，我不想这么做，但是其实你去观察身边的所有人，如果他说我不想的时候，你去问问他。

你不想做，但是你能做到吗？只有能做到，但是不做，这才叫做不想做。而做不到，却说不想，那就是无能，还爱说风凉话。啊，和你说关于朋友圈的第三件事，你要把你最好的最高光的一面展示出来。

那很多人会觉得自己的朋友圈经的太好，真人会不会有所落差呀，这样子是不是有点欺骗啊，其实大可不必这样想。我给你说两个例子啊，我们去超市买方便面，通常包装上的图片都特别的好看，特别精致，牛肉也很大。

然后菜也很新鲜。但是打开之后呢，你会发现里边都是一些干冻的肉干菜干，泡好之后呢，也根本不是封面那个样子，但是你会不会打开包装之后说，哎，骗子，然后呢就扔了，或者说你去找店家理论，哎，你这个照片。

你不会吧，那方便面的厂家会觉得自己用了美化的照片，就是在欺骗客户吗？于是改用一个真实的照片当包装也不会吧。那朋友圈是我们的社交橱窗，那橱窗里应该放什么呢？一定是放我们最好的或者最热卖的。

再或者是最有特色的产品，而店里边放的才是你的全部，那你不会走进一家店，然后说哎为什么你们里面卖的东西没有橱窗里的柜，也没有橱窗里的好，但凡是一个心智健全的人都不会问出来这种话。

那一部电影哪一部分最好看呢？你仔细想一下，一定是宣传片最好。看可能一部电影最精华最重要、最引人注意的内容全都压缩在了那几分钟的预告里，但你还是会花钱去电影院去看整部电影，而不是说在家里看个预告就好了。

所以啊朋友圈里边应该出现的是我们最得意的时候，得到一些成就的时候，以及我们最希望别人记住我们的一些特质。相反，不要有任何的批判抱怨情绪负能量之类的内容。这些内容只适合在小范围亲近私密。

有信任的环境下去释放，而并非公开释放。O总结一下今天的内容，朋友圈其实是你正在努力生活，并且接受社会化的一个象征，你必须重视起来。三个注意点，第一，不要三天可见。朋友圈是你在社交商业街里的橱窗。

你只要是想开门营业，就要用心经营。第二，每个人的朋友圈都是刻意为之的。因为反馈系统会逆向的塑造每一个人的行为。第三，你要展示自己最好的一面，而不要有任何情绪负面的宣泄。
