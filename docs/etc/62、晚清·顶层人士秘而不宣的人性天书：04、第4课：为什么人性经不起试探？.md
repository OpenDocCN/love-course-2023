# 62、晚清·顶层人士秘而不宣的人性天书：04、第4课：为什么人性经不起试探？

亲爱的小伙伴们，今天我们要分享第四课，为什么人性经不起试探？在很多人眼里，人性是个很悬的东西，我们每天都会看到很多张面孔。但是这些面孔下面有几个是真诚的，又有多少是虚伪的呢？😊。

因为人心都是藏在肚子里的，我们看不到也摸不着。所以很多人特别想知道自己身边的人对自己到底有几分真心，于是各种各样试探的手段就应运而生了。😊，有的用言语试探，比如问如果我瘫痪了，你会不会离开我呀？

如果我毁容了，你还爱我吗？但很多人觉得对方的回答未必是真心的，不如自己得来的答案可靠，于是就出现了试探人心。但是我想给大家一句忠告，千万不要去试探人性，一试全局终。给大家讲一个历史故事吧。

大明万历年间有这么一桩案子，有一个很脑残的婆婆，她放着好好的日子不过，非要去考验她儿媳妇，结果考验出人命来了。本来一家四口人过得好好的，但是因为这家的儿子常年在外经商，家里媳妇服侍公婆，这婆婆年纪大了。

疑性病重，总怀疑儿媳妇和自己老公有猫腻。某天晚上，他想到了一个损招，他要试探一下自己这个儿媳妇到底是不是真的忠贞，她偷偷穿上了她老公的衣服，然后趁着夜色就从后面突然抱住了儿媳妇。😊，儿媳妇突然受惊。

黑暗之中又看不清楚，就胡乱踢打，挠了她的脸一下。婆婆松开手跑了，儿媳妇只看见背影，认的是自己公公的衣服，以为是被公公调戏了。第二天，赵儿媳妇就跑回娘家跟自己的亲爹去告状，他说自己被公公欺负了。

他父亲气冲冲的去问罪，却发现亲家公脸上没有伤，以为是自己的女儿撒谎，回家就把女儿给训了一顿。这儿媳妇先是被公公欺负，呃，就是他以为自己被公公欺负了，然后又被亲爹训斥，事情还说不清楚了。

一气之下他就自杀了。😊，女儿自杀以后，他父亲又悔又气，就报了官，这下轮到她公公说不清楚了。后来，邻居提供了重要线索，说看见老太太脸上有伤，官府就把老太太抓来询问，这才真相大白。此案轰动一时。

人称婆奸喜案。抛开狗血因素，就从考验人性的角度来说，这是多么成功的一次考验啊。婆婆得到了她想要的答案，她的儿媳妇也经受住了考验。考验结果证明，儿媳妇确实是忠贞的，儿子是幸福的，自己就不必在疑心了。

自己的老公也是清白的多好呀。但是这结局是死了，儿媳妇关了婆婆。😊，这样去看清别人，也是一种人性。想想看别人在聊天，你是不是很想知道他们是不是在说你呀，想想看，哪怕你和一个人很好。

你是不是很想知道他在背后是怎么对你的。😊，就因为有了这种好奇心和不确定感，就有了试探人性。曾经有位读者很挑衅的问我，婉晴，你这么厉害，你敢不敢找个女人去试探你老公，看看她会不会被其他女人勾引。

会不会爱你一辈子。我撇撇嘴说，我干嘛做这么无聊的事呀？😊，他激将道，你是不敢吧，你怕结局不是你满意的那种吧。没想到你也有不敢的事儿。我冷笑一声说，激将对我无用，你有病，我可没病，要做你就自己去做吧。

把你身边的人统通试探一遍，看看谁对你真心，谁对你假意。这世上唯有太阳和人心不可直视。那为什么不能去试探人性，或者说试探人性会有什么样的后果呢？首先，一个很充实，内心光明磊落且自信的人。

他不会去想着要试探人性，喜欢去试探人性的人，不外乎以下这几种。第一，特别无聊的人，比如有些女孩子恋爱后，每天胡思乱想，男朋友到底爱不爱我，想的多了，觉得自己想想不出一个结果来。

然后就会去找闺蜜啊、姐妹啊，伪装成别的女人去试探自己的男朋友。😊，第二，骨子里就不相信别人的人。比如有的老板害怕自己的下属有异心，然后就会故意去找猎头公司挖他，看看他会不会动心。第三，不自信的人。

他会怀疑身边所有对他好的人，要么别有用心，要么不长久，总想确认一下对方到底是真心还是假意的。也就是说，任何一个人去试探人性之前，其实他已经充满了不信任，充满了恶意的揣测。虽然说是在试探。

但是他潜意识里已经告诉自己，这个人不可信。人的信念是有强大的磁场的，一个人每天想的都是积极快乐的好事儿。那么他很有可能真的会遇到好事儿。一个人天天都去想一些不好的事儿，那悲剧惨剧就真的会发生。

比如一个女人要是天天担心老公出轨，那么这个老公很大概率就会真的出轨。😊，从逻辑上来讲也很好理解，我们每一种想法都会对应相应的行为。当你的想法不断告诉你，这个人不可信，那你的行为中就会对这个人不断怀疑。

连看对方的眼神都是探究的。而人与人之间是有感应的，你那么不信任对方，对方是能够感受到的，但对方感觉到你不信任他，你对他的人品有质疑。他很大的反应就是生气恼火，然后就真的做出那些你害怕的事情来。

这就是试探人心的前提，就已经决定了试探人性的结果。有了恶意的揣测，又有了恶意的试探，结果当然只能得到失望。其次，试探人心是有隐的。当一个人开启了试探别人，他不会只做一次，他会从中得到偷窥的快感。

觉得别人都在明处，自己在暗处，可以看见别人看不见的东西，这也是一种人性。任何阴暗的手段不会只用一次，用过之后就会像打开了的潘多拉魔盒一发而不可收拾。所以当你选择了试探人性，哪怕结果是好的。

你也不会就此收手。得到满意的结果后，你只可能消停一会儿，觉得暂时安心了。但过不了多久，你这个念头又会蠢蠢欲动，想看看他是否还如以前一样真心对你。而且你知道第一次试探的手段对他没用。你只会让试探手段升级。

那么在一次次试探中，对方能扛住几次呢？他迟早有一天会经不起试探，最终让你失望。而试探的人，只有得到了这个失望的结果，才有可能消停。只要结果是好的，就会不停的试探。

这也决定了试探人性为什么最终都是不好的结果。再次，只有不懂人性的人性小白才会去试探人性。大家可以仔细回想一下，那些人性高手会用这么愚蠢的方式去试探别人吗？这么低级的手段，他们哪里看得上呀。

只会觉得弱智且无聊。所以喜欢去试探别人的都是人性小白，对人性一无所知的人。就像刚才说，有些女人喜欢找自己的闺蜜去试探男朋友，结果男朋友没有经受住考验，闺蜜也背叛了她，最后这两人在一起了。

她受到了双重打击，到处哭诉求安慰。是我没有同情心，我觉得这都是吃饱了撑的，至找的。你去试探的时候，就应该明白这种试探会有两个结果，一种是经受住了，一种是失败了，万一失败，可不就是这种结果吗？

哪怕结果没有这么惨，起码有一方已经背叛了你。这种结果是你想要的吗？最后也是最值得我们清醒的一点，哪怕你所有的试探，最终的结果都是如你所愿，你就高枕无忧了吗？不，如果有一天对方发现你曾经这样试探过他。

如此不信任他，虽然他完全经受住了你的考验，但他会觉得自己是个大傻逼。为了你这种不信任他的人，掏心掏肺，简直就是一个笑话。于是，但凡还有点骨气的人，都会选择远离你，不会原谅你。所以如果他没有经受住考验。

是你离开他。如果他经受住了，却发现这是一场考验，是他离开你。可能有人会说，我不让他知道不就可以了吗？天下没有不透风的墙啊。再说了，一个人性小白会有这么周迷的思维吗？被发现也是迟早的事儿，你可以设计场景。

设计考题，但是你无法掌控所有的位置因素，无法把握所有事情的走向，你不知道你能否得到你想要的答案，也不知道即使得到了想要的答案，你是不是还能够圆满的收场。😊，任何考验，不管是情侣之间还是亲戚朋友之间。

都是以透支信任和感情为代价的。你以为你在社区考验别人，其实你也是这个局里的一员。既然人性不能试探，那人性高手会怎么做呢？真正的人性高手从来不会去通过任何手段去试探别人，他很清楚，这事又无聊又没有意义。

所有的试探都指向了人性是恶的，既然都已经定性了，还有什么好试探的呢？所以人性高手会看清人性，并且顺应人性？什么叫看清人性，并顺应人性呢？首先，他从来不追求别人一定要对自己掏心掏肺，始终如意。

每个人都有自私且不为人知的一面，每个人时时刻刻都在变化着，那就是真实的忍性。那种不管我变成什么样，你都会始终如意的对我，哪怕全世界的人都对我不好，你都会对我很好的想法，从来不会出现在人性高手的脑海里。

人性高手知道每个人都有好和坏的一面，哪怕他真的发现对方人品有问题，也不会直勾勾的去试探，而是看破不说破。作为一个极顶聪明的人，不是去试探别人的恶，而是用自己精通人性的本事，让对方用善的一面来对待自己。
