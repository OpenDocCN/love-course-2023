-   五种尬聊方式，看看你中了几个
    1.  自说自话，迫切希望把握社交机会，造成压力，影响对方兴趣。
    2.  躲避关注，紧张自卑，错失了彼此了解的机会。
    3.  查户口提问，集中话题在对方身上，容易让人感到烦躁。
    4.  否定对方，展现自己的高价值，容易给人虚假领导感。
-   沟通优化建议
    1.  提问阶段，找共同点，自然开启话题，避免距离感。
    2.  夸奖开场，避免编辑效应，多角度夸奖对方，增加情感连接。
    3.  清听阶段，使用语气词加上想说的话，促进双方继续交流。
-   通过不同阶段的表达方式，可以让双方的聊天更加兴奋起来，包括选择无视或反驳，反抛话题，评论对方等方式。
    1.  选择无视或反驳，不会导致情况变差，可以让对话更顺畅。
    2.  反抛话题，可以根据对方的兴趣开启新话题，让对话更有趣。
    3.  评论对方，可以通过模糊评价对方的状态，引发更深入的对话。
