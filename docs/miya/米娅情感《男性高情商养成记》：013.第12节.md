# 米娅情感《男性高情商养成记》：013.第12节

hello，大家好，今天我们来聊一下相亲的话题，不知道现在正在听课的你是不是也正在被这些问题困扰。如果是的话，那么今天这节课就对你很重要。在这节课程里，我想带着大家打开思路。

用不同的视角来审视一下相亲这件事儿，或许能够让你接下来的择偶之路变得更顺畅一些。其实说到相亲啊，大部分人内心里还是比较排斥的。那么什么样的情况下大家会去相亲呢？通常啊都是年龄大了，父母着急。

于是被家里安排去相亲。那么为什么大家会排斥相亲这件事儿呢？原因有两点啊，首先是我们的一种主观的自我保护心态，我们会认为出来相亲的人目的性都很强。那么人性就是这样的，虽然我们自己目的性也很强。

但是我们又很不喜欢对方目的性太强，这会让我们紧张，有不安全感，被窥视感。那么在这样的心理状态下，人们就会开启自我保护机制。所以这个时候我们很难感觉到愉悦，也不容易敞开心扉。然而，对方也同样有这样的感觉。

所以啊这就是大家经常抱怨的。相亲为什么总是感觉无法交心。感觉大家都是像例行公事，很难有深入的情感交流。但是如果是我们在不经意间偶然邂逅的一个人，因为我们没有目的性，也没有防备心态，因此更容易产生好感。

更容易产生好奇和愉悦感，以及和想要探索的欲望。于是我们便感觉这是缘分来了，让我们心情愉悦，如沐春风。让大家排斥相亲的另一个原因是待价而沽的心态。我们总觉得自己好像是菜市场里的白菜，标出自己的价格。

看对方愿不愿意买单。对方也是标出他的价格，看看我们能不能接受这种拿到明面上的条件交换，让我们感觉不舒服，我们不得不承认一个事实。就是两个人最终如果能够走向婚姻的话，条件评估这个环节是一定会有的。

双方都要考虑对方的年龄、婚事、家庭背景、经济条件、工作状况等等等等。但是我们不习惯直接，我们更喜欢迂会，喜欢在漫长的相处过程中，逐渐的去用感知对对方进行条件评估。这样让会让我们有一种过程感。

有真正恋爱的感觉，跟买大白菜不一样。其实啊这本质只是先置评估和后置评估的区别罢了。从这个角度来看的话，其实相亲更真诚。我们在生活中想要去结释异性的话，会有很多不同的场景和途径。

比如说学习场景、工作场景、聚会场景、网络社交介绍相亲啊，这些都仅仅是途径，途径和途径之间是没有什么本质区别和好坏之分的。那么有区别的是什么？是人的防御心态？

人们通常对在学习、工作、聚会等场景下结识的异性、防御心态会较弱。但是对网络社交和介绍相亲的异性，防御心态会较强。在不同程度的防御心态下，我们能体验到的愉悦感是不同的。那么对于不同的场景和途径。

其实是有其对应适合的人群的，并不是所有的人都能在学习和工作的场景下结识感兴趣的异性。而网络上的很多虚拟人设也未必真实可靠。因此，从科学的角度看，对于有年龄焦虑和婚姻刚需的人群。

相亲其实是一个非常靠谱的途径。当然了，其他的途径你也可以多轨并行。那么相亲有什么好处呢？首先啊是节约时间成本，也就是机会成本。很多人由于工作忙碌，本身业余时间就不多。

让他们再花时间去各种途径结识异性的成本比较高。每个人的时间和精力都是有限的，一天都是只有24个小时，拿来做一件事就很难兼顾另外一件事。因此，对于工作压力比较大。时间相对紧缩的大龄灵活待婚青年来说。

相亲就是一个短平快的方式，省时省力，不需要先去花很多时间跟对方聊天，反复琢磨，然后精心准备，再小心翼翼的把对方约出来，可以节约掉很多繁琐的环节。其次啊是可以扩大筛选范围。为什么这么说？

因为现在很多人都是不太喜欢社交的，所以认识异性的途径也相对比较有限。再加上很多钢铁直男们本身也不太会跟女性相处，身边的女性资源匮乏，圈子有限。如果在这种现实背景下，想要遇到合适的对象是非常困难的。

那么相亲的好处就体现出来了。可以短时间内让他们有更多的机会结识更多的异性。很多时候我们选不出来喜欢的人是因为样本太少了。举个例子，三个人当中可能不会有一个你喜欢的。那么10个人当中呢，50个人当中呢。

100个人当中呢，这是个概率的问题，只有样本数量足够多，你能选出喜欢的人的可能性才会越大。所以啊对于资源匮乏的男性，相亲就是个扩展资源的好途径。最后啊就是相亲可以降低风险，什么意思？

现在社交网络已经非常发达了，各种交友软件层出不穷，质量也良莠不齐。很多有社交恐惧或者身边异性资源短缺的男性会愿意到交友软件里结识女性。这里就有风险了。首先大家都是陌生人。

对对方的基本资料和个人信息都是对方说出来的，没有人去核实其真实性，这也给一些别有用心的人提供了机会。很多钢铁直男们又恰恰缺乏这方面的判断力，往往很容易被迷惑，最后造成一些不必要的损失。除了金钱之外。

更重要的是可能会因此消耗很多的时间以及投入感情之后带来的心理创伤。当然啊我们不能一棒子打死所有的人，也并不是所有交友软件认识的人都不靠谱。当然也有靠谱的。那么问题是你是否具备强大的筛选能力。

能把靠谱的人筛选出来，把不靠谱的人屏蔽掉。现实生活中大部分的人是不具备的。因此啊这个时候相亲的优势就体现出来了，相亲就是有人先帮你做了筛选，能够在最开始就掌握更多的真实信息，大大降低了交友的风险。

那么你说相亲有没有真感情呢？我的答案是有的。相亲只是一种途径，并不是一种结果，在我众多的粉丝当中，也不乏有人在相亲的时候，就对对方一见钟情的？所以啊大家不必要去排斥相亲，你的心态很重要。

并且你的心态从某种意义上来说，也会影响你相亲的成功概率。为什么这么说。当你是一个消极排斥的心态的话，对方是能感受到的，对方会感受到你的应付了事和不耐烦，这首先就会拉低对方对你的好感了。

即使对方对你有点意思，也可能会因为你的消极态度而望而却步了。反而如果你是一种积极正面的心态的话，对方也是能感受到的，会增加好感，增加对方想要进一步了解你的概率。即使相亲不成，大家大不了当个朋友。

也许对方身边有合适你的人，还会再帮你介绍。这就是一种正向积极的磁场。也许啊久而久之，你就离脱单不远了。那么不管是什么渠道，两个人在一起的，感情都是要经营的。如果不懂得经营的话，就算是青梅竹马。

婚后的感情也很可能是一地鸡毛。我们只要正确看待每一种交友途径，并且努力提高自己的情感智力，相信最终一定会找到合适的另一半。
